## 第一层：直觉驱动（经验 + 市场理解）

**从"为什么会赚钱"出发**，而不是数据挖掘

- **行为金融学**：投资者过度反应/反应不足 → 动量/反转因子
- **资金结构**：机构抱团、游资炒作、北向持续买入 → 资金流向因子
- **市场微观结构**：换手率异常 = 筹码换手；涨停后缩量 = 锁仓

> Z 哥的"白线黄线/量价配合"本质就是这一层——对 A 股资金行为有直觉理解，再把直觉转化为计算规则。

**当前系统已有的第一层因子**：
- 白线（EMA of EMA 10,10）、黄线、KDJ(9,3,3)
- 量比、缩量标志、N 型结构、前低保护
- ATR14、近 20 日高低点

## 第二层：学术论文 → 实盘验证

**流程**：读论文 → 复现 → A 股回测 → 看 IC 稳定性（目标 Rank IC > 0.04）

### 经典因子体系

| 因子类别 | 代表因子 | A 股有效性 |
|---------|---------|-----------|
| 市值 | 小市值因子（SMB） | ✅ 显著 |
| 价值 | PB、PE、PS | ✅ 中等 |
| 动量 | 12 个月动量 | ⚠️ 较弱 |
| 反转 | 1 个月反转 | ✅ 显著 |
| 盈利 | ROE、毛利率 | ✅ 中低频 |
| 非流动性 | Amihud ILLIQ = \|ret\|/volume | ✅ A 股有效 |

### A 股特色学术因子
- **日历效应**：节前效应、月初效应（A 股显著）
- **打板/涨停因子**：连板天数、涨停后首日表现
- **北向资金跟踪**：陆股通净买入量变化率
- **融资融券**：融资余额变化率（情绪指标）

## 第三层：遗传规划 / 符号回归（自动因子挖掘）

WorldQuant、Two Sigma 最爱的方法。

```python
# 从基础算子 + 原始数据中，自动搜索有效公式
operators = [+, -, *, /, ts_mean, rank, delta, log, sign, ts_std...]
data = [open, high, low, close, volume, amount, turnover...]

# 遗传算法生成候选因子示例：
factor_A = rank(delta(close, 5)) / ts_std(volume, 20)
factor_B = log(amount / ts_mean(amount, 10)) * sign(close - ma20)
factor_C = (high - close) / (high - low)  # 上影线因子
```

**工具**：
- Qlib 表达式引擎：`Ref($close, 5) / $close` 直接配置
- AlphaNet（CNN）：用神经网络代替遗传算法搜索特征交互
- WorldQuant WebSim：全球最大量化因子搜索平台

**注意**：自动挖掘的因子容易过拟合，**必须有逻辑解释**，否则样本外快速失效。

## 第五层：因子的"工业化"管理

顶级量化机构不只是挖因子，还建立完整的因子管理体系：

1. **因子正交化**：去掉行业/市值暴露，提取"纯 alpha"（防止因子实际是 beta）
2. **因子半衰期测试**：IC 衰减多快？衰减快的因子需要高换仓频率
3. **因子相关性检查**：避免因子堆叠（本质是同一个信号被重复计入）
4. **因子暴露控制**：单因子权重不超过阈值（防止单一风格崩塌）
5. **因子库版本管理**：记录每个因子的发现时间，防止回测过拟合

### Z 哥对称战法因子化思路

Z 哥"对称战法"：价格在前期关键位置（前高/前低/颈线）对称位置容易形成支撑/阻力。

**因子化方向**：
```python
# 对称支撑因子
symmetric_support = 2 * pivot_low - pivot_high  # 前低的对称位

# 当前价格距对称支撑的距离
dist_to_symmetric = (close - symmetric_support) / close

# 对称位突破因子（0/1）
symmetric_breakout = (close > symmetric_support) & (close[-1] < symmetric_support)
```

## 总结：因子挖掘核心原则

1. **因子必须有逻辑支撑**，纯数据挖掘容易过拟合
2. **A 股特色**：反转因子 > 动量因子；资金流向因子有效；小市值效应显著
3. **优先低成本因子**：用本地已有数据能计算的先上，akshare 另类数据其次
4. **因子质量标准**：Rank IC > 0.04，IC IR > 0.5，才算有效因子
5. **因子越多不越好**：高相关因子要去重，保留正交信息
