## 各域代表性论文摘要

### rec-sys
- **HSTU (Meta)**: 将推荐重新定义为序列生成任务，万亿参数级 Transformer 在工业规模验证 Scaling Law，统一召回和精排。这是生成式推荐的里程碑工作。
- **PROMISE**: 首次将过程奖励模型（PRM）引入推荐领域，实现测试时计算扩展，类似 o1 推理范式在推荐中的应用。
- **Agentic RecSys**: 前瞻性综述，从 pipeline 推荐走向 Agent 推荐系统，预示下一代推荐架构方向。

### ads
- **EGA-v2**: 端到端生成式广告框架第二代，Semantic ID + RQ 量化实现广告检索与排序统一。
- **MGOE**: 宏观专家图架构，十亿级多任务推荐中的 Gumbel-Softmax 结构学习，实现专家间宏观级路由。
- **IDProxy (小红书)**: 多模态 LLM 生成代理嵌入解决冷启动，端到端对齐 ID 嵌入空间。

### search
- **OneSearch**: 电商搜索端到端生成式统一框架，检索/排序/重排一体化，消除级联误差。
- **KaLM-Embedding-V2**: 0.5B 参数紧凑嵌入模型，渐进多阶段训练 + Focal 重加权，MTEB 超越 3-26 倍大模型。

### llm-infra
- **Search-R1**: 用 RL 训练 LLM 推理并调用搜索引擎，DeepSeek-R1 思路在搜索工具使用上的延伸。
- **PLENA**: 硬件软件协同设计优化长上下文 Agentic LLM 推理，解决 KV Cache 和注意力效率瓶颈。

## 关键技术趋势观察

1. **生成式范式全面渗透**：从推荐（HSTU/MTGR/UniGRec）到广告（EGA-v2/GPR）到搜索（OneSearch/OneSug），生成式方法正在统一传统的多阶段 pipeline。核心驱动力是消除级联误差和实现端到端优化。

2. **Scaling Law 在推荐/广告中验证**：HSTU 验证了万亿参数级推荐模型的 Scaling Law，MTFM 验证了模型规模 3 倍增长仍有正收益。这意味着 "大力出奇迹" 在推荐领域同样适用。

3. **测试时计算与推理增强**：PROMISE 将 PRM 引入推荐，Search-R1/TongsearchQR 用 RL 训练搜索推理。这是 o1/R1 推理范式在应用领域的落地前沿。

4. **Agent 化趋势**：Agentic RecSys 综述 + Multi-Agent Workflow 生产架构，预示下一代系统将从静态模型走向动态 Agent 编排。

5. **冷启动的 LLM 解法**：IDProxy 用多模态 LLM 生成代理嵌入，ICL-Bandit 用 Bandit 优化 LLM 上下文学习，LLM 正在成为冷启动问题的通用解法。
