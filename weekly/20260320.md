## 今日 Top 3 技术点（资深工程师视角）

1. **LLM 推荐系统的新范式**（rec-sys）
   LLM Universal Recommendation Learners 等论文说明 LLM 作为推荐 backbone 已不只是 feature encoder，而是统一的语义推理器。工业落地挑战：latency（需要 speculative decoding + KV cache 压缩）和 collaborative signal 的注入（纯 LLM 缺乏 CF 信号，需要混合架构）。

2. **First-Price 拍卖下的出价策略**（ads）
   2019 年 Google 切换到 First-Price Auction 后，出价策略从 truthful bidding（VCG 下最优策略）变成了需要学习最优 shading 系数的 RL 问题。论文关注 non-stationary 环境、budget constraints 下的 regret minimization。对工程的启示：oCPC 的 PID 控制器是一种特殊的 bid shading 机制。

3. **KV Cache 压缩是 LLM 推理的核心战场**（llm-infra）
   KVSharer、ZeroMerge 等工作表明：KV Cache 在相邻层之间存在高度相似性，可以共享或压缩至 1/20。结合 MegaScale-Infer 的 Expert Parallelism，MoE + KV Cache 压缩是当前 LLM 规模化部署的主流方向。DeepSeek-V3 和 Qwen3 都在这两个维度深度优化。
