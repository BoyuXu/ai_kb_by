├─ LambdaRank / LambdaMART

└─ 生成式排序模型

├─ 排序层（精排）

└─ 线上部署
```

```
数学基础

├─ 数据增强和重采样

├─ ESMM（全空间多任务）
└─ PAL（位置感知学习）

因果推断
