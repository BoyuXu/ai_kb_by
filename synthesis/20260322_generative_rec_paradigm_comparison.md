# 生成式推荐范式对比：自回归 vs 扩散模型

**一句话**：生成式推荐把「找商品」变成「写文章」——但写文章有两种方式：逐字写（自回归）和先画草稿再慢慢雕琢（扩散），各有各的优势。

**类比**：你要写一封推荐信。自回归方式：从「亲爱的」开始，逐字逐句写到结尾，写完一个字才能写下一个字。扩散方式：先随手涂鸦一封信（加噪），然后一遍遍修改（去噪），最终得到完整的信。扩散允许你改变任何位置，而自回归只能往后写。

---

## 两种范式对比

| 维度 | 自回归（TIGER, SASRec） | 扩散（DiffGRM） |
|------|------------------------|----------------|
| 生成方式 | 逐 token 串行生成 | 并行迭代去噪 |
| 上下文 | 单向（只看已生成的） | 双向（全局感知） |
| 曝光偏差 | 有（训练用真实 token，推理用预测 token） | 无（去噪过程不依赖自身历史） |
| 多样性 | 确定性强，多样性依赖 Beam Search | 随机性内置，多样性自然更高 |
| 推理延迟 | O(L) 串行解码步骤 | O(T) 去噪步骤（T 通常 10-50） |
| 训练收敛 | 快（2× 更快） | 慢 |
| 目前精度 | 更高（工业主流） | 学术追赶中（+8.3% HR@10） |

---

## DiffGRM 的核心思路

```
传统 DDPM（图像生成）：
  图片 → 加高斯噪声（T步） → 纯噪声
  纯噪声 → 去噪（T步，每步用 UNet）→ 图片

DiffGRM（生成式推荐）：
  Semantic ID 嵌入 → 加高斯噪声 → 噪声向量
  噪声向量 → 条件去噪（条件：用户历史编码） → Semantic ID 嵌入
  嵌入 → ANN 检索最近邻 → 候选商品
```

**关键细节**：
- 在连续嵌入空间扩散（不是离散 token），更稳定
- 条件：用户历史序列的 Transformer 编码向量
- 双向 Attention：扩散 Transformer 可以看到所有噪声位置
- 课程扩散：训练时先学「大噪声去噪」（粗粒度），再学「小噪声去噪」（细粒度）

---

## 为什么扩散没有曝光偏差？

自回归的曝光偏差（Exposure Bias）：
```
训练时：input = [真实历史, 真实token_1, ...], predict token_2
推理时：input = [真实历史, 预测token_1, ...], predict token_2
                              ↑ 预测的，可能有误差
```
误差会随序列长度累积。

扩散模型没有这个问题：
- 训练时：直接预测「从噪声到原始嵌入的映射」
- 推理时：同样的映射，只是从随机噪声开始
- 没有「先预测的结果作为下一步输入」的链式依赖

---

## 什么场景用扩散，什么场景用自回归？

| 场景 | 推荐方案 | 理由 |
|------|---------|------|
| 推荐多样性要求高（音乐、视频） | 扩散 | 内置随机性 |
| 推荐精度优先（广告、高转化） | 自回归 | 精度更高，可控 |
| 超长历史序列（>1000 items） | 自回归 + GEMs（今日） | 扩散的双向 Attention 计算量更大 |
| 冷启动+内容丰富 | 扩散 | 连续嵌入空间更好利用内容特征 |

## 技术演进脉络

```
2020 SASRec：Transformer 自回归序列推荐
    ↓ 离散 item_id → 词汇量爆炸
2022 TIGER/P5：Semantic ID + 自回归，生成式推荐
    ↓ 曝光偏差 + 单向上下文限制
2024 DiffGRM：扩散模型替代自回归，消除曝光偏差
    ↓ 两条路线并行发展
2025 可变长度 Semantic ID（与自回归兼容）
     Long-sequence（GEMs）提升自回归效率
```

## 面试考点

**Q：生成式推荐和传统推荐的核心区别是什么？**  
答：传统推荐（双塔+ANN）是判别式的：给定 user 和 item，预测相关性分数。生成式推荐是生成式的：给定 user，直接生成目标 item 的标识（Semantic ID），通过解码而非检索得到候选。优势：能处理超大候选集（不需要向量化所有 item），更自然地融合内容语义；劣势：训练更复杂，延迟更高（自回归解码）。

**Q：DiffGRM 的连续空间扩散 vs 离散扩散有什么区别？**  
答：离散扩散（D3PM）在 token 空间加噪（随机替换 token），但 Semantic ID 的码字空间不像文字那样有丰富的替换语义，收敛不稳定。连续扩散在 embedding 向量上加高斯噪声，扩散过程平滑且数学性质好（DDPM 理论保证），最后通过 ANN 检索回离散 ID，两全其美。
