## 一、广告冷启动

### 1.1 冷启动的三类场景

| 场景 | 描述 | 难点 |
|------|------|------|
| 新广告 | 新上线广告无历史CTR数据 | Embedding 无法学习，CTR预估不准 |
| 新广告主 | 新开户广告主无投放记录 | 无法估计出价合理性和创意质量 |
| 新用户 | 新注册用户无行为历史 | 个性化排序失效 |

### 1.2 新广告冷启动方案

**方案 1：IDProxy 渐进迁移（小红书）**

核心思想：用相似的热门广告 ID 作为代理，渐进替换为自身 ID。

$$
\text{ProxyIDs} = \text{TopK}_{j \in \mathcal{H}}[\text{sim}(e_i^{\text{content}}, e_j^{\text{content}})]
$$

渐进迁移公式：

$$
e_{\text{final}}(t) = (1 - e^{-\mu N_{\text{clicks}}}) \cdot e_{\text{proxy}} + e^{-\mu N_{\text{clicks}}} \cdot e_{\text{id}}
$$

当点击累积到一定量，自身 ID embedding 逐渐替代代理 ID。

为什么不直接用内容 Embedding？
- 保持模型架构统一（无需修改 Embedding Table 结构）
- 代理 ID 携带协同过滤信息（热门广告的用户行为统计），纯内容特征没有

**方案 2：LLM 语义特征增强**

用 LLM 从广告文案/图片中提取语义特征作为冷启动信号：
- 广告标题 → LLM embedding → 作为辅助特征
- 结合 CLIP 提取视觉-文本对齐特征
- 在热门广告上预训练，迁移到新广告

**方案 3：探索-利用策略**

| 策略 | 公式 | 适用场景 |
|------|------|---------|
| UCB | $\text{UCB}(a) = \hat{\mu}_a + \sqrt{\frac{2\ln t}{n_a}}$ | 展示次数少的新广告获得更多曝光 |
| Thompson Sampling | 从 $\text{Beta}(\alpha_a, \beta_a)$ 采样 | 更自然的探索，收敛更快 |
| $\epsilon$-greedy | 以 $\epsilon$ 概率随机展示 | 最简单，但效率低 |

工业实践：新广告上线前 48h 进入探索期，用 Thompson Sampling 分配 5-10% 的流量。

### 1.3 新用户冷启动

- **注册行为特征**：设备型号、地区、来源渠道、注册时段
- **用户相似度**：找历史中最相似的 Top-K 用户，用其 CTR 统计作为代理
- **Early Signal 快速更新**：前 24-72h 行为信号每小时刷新预测

### 1.4 跨用户挖掘（PRECTR-V2）

冷启动用户自身信号少，相似用户群的行为统计提供代理信号：

$$
h_u^{\text{cross}} = \text{CrossAttention}(e_u, \{e_{u'}\}_{u' \in \text{similar}})
$$

Cross-user attention 学习"相似用户对同一广告的反应"。

## 二、偏差治理三部曲

### 2.1 位置偏差（Position Bias）

**现象**：第 1 位的点击率 = 第 3 位的 2.5-3x（工业 A/B 实验数据）。如果不纠正，模型学会"展示在第一位就预测高 CTR"的捷径。

**IPS（逆倾向加权）纠正**：

$$
w_i = \frac{1}{P(\text{examined} \mid \text{position}_i)}
$$

examination probability 通过随机化实验估计（随机打乱排名，观测各位置点击率）。

**PAL（Position-Aware Learning）**：

$$
P(\text{click}) = P(\text{examine} \mid \text{pos}) \times P(\text{click} \mid \text{examine}, \text{ad})
$$

训练时建模位置影响，推理时去除位置因子，得到无偏 CTR。

**Shallow Tower（浅层塔）**：
- 主模型学广告质量
- 浅层塔学位置偏差
- 推理时只用主模型，丢弃浅层塔

### 2.2 选择偏差（Selection Bias / SSB）

**现象**：CVR 模型只在点击样本上训练，但推理时在全曝光空间使用。训练分布 ≠ 推理分布。

**ESMM 系列解决方案**：

$$
P(\text{CTCVR}) = P(\text{CTR}) \times P(\text{CVR})
$$

在全曝光空间建模 CTCVR，间接约束 CVR。详见 [[synthesis/ads/01_CTR_CVR预估与校准全景|CTR/CVR 预估与校准]]。

**ESCM² 因果纠偏**：

$$
\hat{\mathcal{L}_{\text{DR}} = \hat{\mathcal{L}_{\text{DM}} + \sum_{i:O_i=1} \frac{1}{p_i^{\text{CTR}}}(\ell_i^{\text{actual}} - \ell_i^{\text{imputed}})
$$

双重鲁棒估计器：outcome 模型和 propensity 模型双重保险。

### 2.3 兴趣坍塌（Interest Collapse / Filter Bubble）

**现象**：用户点击了几次运动鞋广告 → 系统持续推运动鞋 → 用户看不到其他品类 → 用户兴趣被"锁死"。

**DIN 解决方案**：

$$
\text{Attention}(q, k_i) = \text{softmax}(W \cdot [q; k_i; q \odot k_i])
$$

DIN 的 target attention 让模型关注与当前广告相关的历史行为，而非所有历史，减少兴趣单一化。

**多样性约束方案**：

| 方法 | 思路 |
|------|------|
| DPP | 行列式点过程，同时优化质量和多样性 |
| MMR | 最大边际相关性，迭代选择与已选结果不同的 item |
| 兴趣探索预算 | 预留 10-20% 展示位给非主兴趣品类 |

### 2.4 延迟转化偏差

**现象**：点击后 7 天才下单的转化被标为负样本（假阴性）。

**处理方案**：
1. 等待窗口法：点击后等 7 天再确认标签（简单但浪费数据）
2. Elapsed-Time Model：将"点击后经过多久"作为特征，预测最终转化概率
3. Fake Negative Calibration：短期标为负样本，事后回补正样本
4. 多窗口建模：同时训练 1d/7d/30d CVR

## 三、偏差治理的统一视角

### 三类偏差对比

| 偏差类型 | 来源 | 影响 | 核心解法 |
|---------|------|------|---------|
| 位置偏差 | 高位展示更多曝光 | CTR 虚高 | IPS/PAL/Shallow Tower |
| 选择偏差 | 只有点击样本有 CVR 标签 | CVR 训练有偏 | ESMM/ESCM²/IPW+DR |
| 兴趣坍塌 | 正反馈循环 | 用户兴趣锁死 | DIN/DPP/多样性探索 |
| 延迟转化 | 转化滞后于点击 | 假阴性训练 | 窗口法/Elapsed-Time |

### 因果推断视角的统一框架

所有偏差本质上都是**混杂因素**导致的训练/推理分布不一致：
- 位置偏差：位置是混杂因素（影响曝光和点击）
- 选择偏差：点击行为是混杂因素（影响 CVR 样本可见性）
- 兴趣坍塌：推荐策略是混杂因素（影响用户看到什么）

统一解法思路：**因果推断**（IPS/IPW/DR/Do-Calculus）+ **随机化实验**（A/B 测试验证因果效应）。

## 四、面试高频 Q&A（12 题）

**Q1: 新广告冷启动为什么用代理 ID 而非直接用内容 Embedding？**
保持模型架构统一（无需修改 Embedding Table 结构），代理 ID 携带协同过滤信息（热门广告的用户行为统计），纯内容特征没有这些信息。

**Q2: IDProxy 的渐进迁移公式含义？**
$e_{\text{final}} = (1-e^{-\mu N}) \cdot e_{\text{proxy}} + e^{-\mu N} \cdot e_{\text{id}}$。初始阶段代理 ID 主导（点击少），随着点击累积，指数衰减项趋零，自身 ID 逐渐替代代理。

**Q3: 位置偏差怎么量化？**
随机化实验：将广告随机排列（忽略排序分数），统计各位置的点击率。第 1 位 CTR 通常是第 3 位的 2.5-3x。用 $P(\text{examine}|\text{pos})$ 作为 IPS 权重。

**Q4: PAL 和 Shallow Tower 的区别？**
PAL 将 $P(\text{click})$ 分解为 $P(\text{examine}|\text{pos}) \times P(\text{click}|\text{examine},\text{ad})$，推理去位置。Shallow Tower 用浅层网络单独建模位置偏差，推理时丢弃。原理类似，Shallow Tower 工程上更灵活。

**Q5: ESMM 如何解决选择偏差？**
通过 $P(\text{CTCVR}) = P(\text{CTR}) \times P(\text{CVR})$ 在全曝光空间建模。CVR 塔通过 CTCVR loss 间接获得全空间训练信号，无需在有偏的点击空间单独训练。

**Q6: ESCM² 比 ESMM 好在哪？**
ESMM 乘法分解假设过强，ESCM² 用反事实框架引入 IPW 去偏（用 $1/p^{\text{CTR}}$ 重加权），DR 估计器双重鲁棒（outcome/propensity 任一正确即可无偏）。

**Q7: 兴趣坍塌的检测指标？**
用户兴趣品类覆盖度（Top-K 推荐中品类数）、用户会话内品类重复率、长期留存率。兴趣坍塌通常表现为短期 CTR 不降但长期留存下降。

**Q8: DPP 如何同时保证质量和多样性？**
DPP 选择概率 $P(S) \propto \det(L_S)$，其中 $L$ 的对角线编码质量，非对角线编码相似度。行列式在高质量+低相似度时最大。

**Q9: 延迟转化处理中 Elapsed-Time Model 的原理？**
将"点击后经过多久"作为特征输入模型，预测"在给定已过时间条件下最终转化的概率"。类似生存分析，对未转化但未到归因窗口的样本给出概率估计，而非简单标为负样本。

**Q10: 触发式广告（DAIAN）与普通展示广告的建模差异？**
触发广告有即时意图信号（用户正在搜索），需实时推断意图向量并与广告对齐；普通展示广告依赖历史兴趣建模。触发广告的 CTR 通常更高但竞争也更激烈。

**Q11: 多路相关性（PRECTR-V2 跨用户挖掘）如何提升冷启动广告效果？**
冷启动用户自身信号少，相似用户群的行为统计提供代理信号。Cross-user attention 学习"相似用户对同一广告的反应"，本质是用群体智慧弥补个体数据不足。

**Q12: 偏差治理的统一因果视角？**
所有偏差都是混杂因素导致训练/推理分布不一致。位置是曝光-点击的混杂，点击是 CVR 可见性的混杂，推荐策略是兴趣观测的混杂。统一解法：因果推断（IPS/IPW/DR）+ 随机化实验验证。
