# AutoAgent: A Fully-Automated and Zero-Code Framework for LLM Agents

- **Date**: 2025-02
- **Domain**: LLM-Infra/Agent Framework
- **URL**: https://arxiv.org/abs/2502.05957
- **Code**: https://github.com/HKUDS/AutoAgent

## 核心贡献

AutoAgent让用户仅通过自然语言即可创建和部署LLM Agent，无需编程技能。解决了全球仅0.03%人口具备必要编程技能的限制。

## 架构组件

1. **Agentic System Utilities**: 系统级工具集
2. **LLM-powered Actionable Engine**: LLM驱动的可执行引擎
3. **Self-Managing File System**: 自管理文件系统
4. **Self-Play Agent Customization**: 自博弈Agent定制模块

## 实验结果

- GAIA benchmark上超越现有SOTA多Agent方法
- RAG相关能力持续优于其他LLM方案
