# Reinforcement Learning Meets Large Language Models: A Survey

- **Type**: Survey
- **URL**: https://arxiv.org/abs/2509.16679
- **Date**: 2026-04

## 核心贡献

全面综述RL在LLM全生命周期中的应用，特别聚焦RLVR（可验证奖励的强化学习）。

## 关键技术

- **RLVR**: 使用程序化验证/证明检查的客观奖励信号
- **全生命周期覆盖**: 预训练 → 微调 → 对齐 → 推理
- **框架整理**: 开源RL fine-tuning框架、数据集、benchmark系统梳理

## 核心洞察

- RLVR是提升LLM推理能力的关键创新
- RL在预训练、后训练、推理阶段均有重要应用
- 可验证奖励避免了reward hacking问题

## 面试考点

RLHF vs RLVR区别，PPO/DPO/GRPO对比，reward model设计，RL在LLM中的应用场景。
