# Cognee: Knowledge Engine for AI Agent Memory

- **Type**: Open-Source Engine
- **URL**: https://github.com/topoteretes/cognee

## 核心技术

- 六阶段pipeline：分类→权限→分块→LLM实体关系提取→摘要→embedding+图
- memify特性：修剪过时节点、强化频繁连接、基于使用信号重新加权
- 向量搜索+图数据库+认知科学方法结合
- \$7.5M种子轮(2026-02)，OpenAI和FAIR创始人支持

## 面试考点

Agent记忆架构，知识图谱构建pipeline，自适应检索策略。
