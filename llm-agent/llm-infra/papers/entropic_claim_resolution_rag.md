# Entropic Claim Resolution: Uncertainty-Driven Evidence Selection for RAG

- **Type**: Research Paper
- **URL**: https://arxiv.org/abs/2603.28444
- **Date**: 2026-04

## 核心贡献

将RAG推理重构为熵最小化问题，用期望熵递减（EER）指导证据选择。

## 关键技术

- **决策论证据选择**: 基于EER最大化选择证据，而非仅相关性
- **CSGR++集成**: 无需微调，即插即用多策略检索
- **证据充分性判断**: 数学化机制判断证据是否足够

## 核心洞察

- 从相关性最大化转向不确定性最小化
- 处理冲突证据场景的关键突破
- 知识密集型任务可靠性提升

## 面试考点

RAG证据选择策略，信息论在NLP中的应用，不确定性量化方法。
