#### 1. **Memori: A Persistent Memory Layer for Efficient, Context-Aware LLM Agents**
- **发布日期：** 2026-03-20
- **作者：** Luiz C. Borro, Luiz A. B. Macarini, Gordon Tindall, Michael Montero, Adam B. Struck
- **核心创新：**
  - 提出持久化内存层（Persistent Memory Layer）架构
  - 支持 context-aware 推理，避免 context window 溢出
  - 可自适应管理长期内存
- **应用场景：** 长期 Agent 部署（月度/年度尺度）
- **技术方向：** Tier-1 基础论文 ✅

#### 2. **All-Mem: Agentic Lifelong Memory via Dynamic Topology Evolution**
- **发布日期：** 2026-03-19
- **作者：** Can Lv, Heng Chang, Yuchen Guo, Shengyu Tao, Shiji Zhou
- **核心创新：**
  - **动态拓扑进化** —— 内存图结构随交互自适应演变
  - 支持连续学习（lifelong learning）
  - Token 高效，检索延迟低
- **关键概念：**
  - 内存节点 = 交互事件
  - 边权重 = 相关性
  - 拓扑变化 = 时间与语义演变
- **应用场景：** 多月/多年长期部署
- **面试相关性：** ⭐⭐⭐⭐⭐ （极高，动态性强）
- **笔记状态：** 待深入阅读

## 🎯 **关键发现总结**

### **三大技术方向确认**

1. **持久化内存层（Persistent Memory Layers）**
   - 代表作：Memori
   - 核心问题：Context window 限制
   - 解决方案：外部存储 + 智能检索
   - **推荐系统对标：** 用户历史行为库

2. **动态拓扑进化（Dynamic Topology Evolution）**
   - 代表作：All-Mem
   - 核心问题：内存随时间演变
   - 解决方案：图结构动态更新
   - **推荐系统对标：** 兴趣图/知识图的增量演变

3. **结构化检索（Structured Retrieval）**
   - 代表作：Conversation Tree, GSEM
   - 核心问题：混乱的扁平化上下文
   - 解决方案：树形/图形索引
   - **推荐系统对标：** 层级化推荐（候选生成 → 排序）

### **与推荐系统的交叉点**

| 问题 | Agent 内存 | 推荐系统 | 技术转移 |
|------|-----------|--------|--------|
| 用户行为表示 | 交互事件 | 行为序列 | 同构 |
| 长期依赖 | lifelong memory | 历史行为 | All-Mem 思想 |
| 实时更新 | 在线学习 | 强化学习 | 反馈融合 |
| 检索效率 | 毫秒级 | 毫秒级 | 倒排索引/向量索引 |
| 个性化 | 用户偏好向量 | 用户向量 | User Preference Modeling |
