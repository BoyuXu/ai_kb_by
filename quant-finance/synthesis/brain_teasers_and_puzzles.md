## Part 1：20道经典概率题

#### 题1：掷两个骰子，点数之和为7的概率

**题目**：同时掷两个公平骰子，求点数之和恰好为7的概率。

**思路**：枚举所有和为7的组合，除以总样本空间36。

**解答**：和为7的组合有 (1,6)(2,5)(3,4)(4,3)(5,2)(6,1) 共6种，总共 6×6=36 种等可能结果。

**答案**：P = 6/36 = 1/6 ≈ 16.67%

**面试官考察**：基本计数能力，能否快速枚举而不遗漏。

#### 题2：抽屉里黑白袜子问题

**题目**：抽屉里有12只黑袜子和8只白袜子，完全黑暗中随机取袜子，至少取几只才能保证有一双同色的？

**思路**：鸽巢原理——2种颜色，取3只必有两只同色。

**解答**：只有黑、白两种颜色。最坏情况下前两只各取一种颜色，第3只无论取什么都与前两只之一同色。注意题目问的是"保证"，所以要考虑最坏情况。

**答案**：至少取3只。

**面试官考察**：鸽巢原理的应用，区分"保证"与"可能"。

#### 题3：生日问题

**题目**：一个房间里有N个人，至少两人同一天生日的概率超过50%，N最小是多少？

**思路**：计算"所有人生日都不同"的补事件概率。

**解答**：P(至少两人同天) = 1 - P(所有人不同) = 1 - (365/365)(364/365)(363/365)...(（366-N）/365)。当 N=23 时，P ≈ 50.73% > 50%。直觉上23远小于365/2，这就是生日悖论的核心——碰撞概率增长速度远快于线性，因为比较的是 C(N,2) 对。

**答案**：N = 23

**面试官考察**：补事件思维、对组合爆炸的直觉。

#### 题4：蒙提霍尔问题（三门问题）

**题目**：三扇门后一扇有车两扇有羊。你选一扇门后，主持人打开另一扇有羊的门，问你是否换门？

**思路**：初始选中车的概率1/3，未选中的概率2/3全部转移到剩余那扇门。

**解答**：不换中奖概率 = 1/3。换门中奖概率 = 2/3。关键在于主持人知道门后的东西且必定开羊门，这个额外信息使概率不再均匀。可以用贝叶斯公式严格证明：P(车在剩余门|主持人开了羊门) = (2/3)×1 / ((1/3)×1 + (2/3)×1) = 2/3。

**答案**：应该换门，换门后获奖概率为 2/3。

**面试官考察**：条件概率与贝叶斯推理，是否能清楚解释信息更新。

#### 题5：赌徒破产问题

**题目**：赌徒有 a 元，每局以概率 p 赢1元、概率 q=1-p 输1元。目标赢到 N 元，求破产概率。

**思路**：建立递推方程，利用边界条件求解。

**解答**：设 P(i) 为从 i 元出发最终破产的概率。P(i) = p·P(i+1) + q·P(i-1)，边界 P(0)=1, P(N)=0。当 p≠q 时，解为 P(i) = ((q/p)^N - (q/p)^i) / ((q/p)^N - 1)；当 p=q=1/2 时，P(i) = 1 - i/N。

**答案**：公平赌局(p=0.5)下，破产概率 = 1 - a/N。不公平赌局见上式。

**面试官考察**：随机过程基础、差分方程求解能力。

#### 题6：醉汉随机游走回到原点的概率

**题目**：一维整数格上，醉汉从原点出发，每步等概率向左或向右移动一步。他最终回到原点的概率是多少？

**思路**：一维随机游走是常返的（Polya定理）。

**解答**：一维对称随机游走中，P(回到原点) = 1。虽然回到原点的期望时间是无穷大，但以概率1最终会回到原点。推广：二维也是常返的（P=1），但三维不是（P ≈ 0.3405）。可以用生成函数证明：回到原点的概率 = 1 - 1/G(0)，其中 G(0) 为 Green 函数，一维情况下 G(0) 发散，故概率为1。

**答案**：概率为 1（几乎必然回到原点）。

**面试官考察**：随机游走基础知识、Polya定理、维度对常返性的影响。

#### 题7：连续掷硬币首次出现连续两个正面的期望次数

**题目**：不断掷公平硬币，首次出现连续两个正面（HH）时停止，求期望投掷次数。

**思路**：用状态机建模——状态0（初始）、状态1（上一次H）、状态2（HH，终止）。

**解答**：设 E0 为从状态0出发的期望次数，E1 为从状态1出发的期望次数。E0 = 1 + (1/2)E1 + (1/2)E0，E1 = 1 + (1/2)·0 + (1/2)E0。由第二个方程 E1 = 1 + E0/2，代入第一个：E0 = 1 + (1/2)(1 + E0/2) + E0/2 = 1 + 1/2 + E0/4 + E0/2，化简 E0 - 3E0/4 = 3/2，E0 = 6。

**答案**：期望投掷次数 E = 6。

**面试官考察**：Markov链状态转移、递推方程建模能力。

#### 题8：错排问题

**题目**：N个人随机排列，没有任何一个人在自己原来位置上的概率是多少？

**思路**：容斥原理计算错排数 D(N)，再除以 N!。

**解答**：D(N) = N! · Σ_{k=0}^{N} (-1)^k / k!。所以概率 P = D(N)/N! = Σ_{k=0}^{N} (-1)^k / k! ≈ 1/e ≈ 0.3679。当 N 较大时，该概率迅速收敛到 1/e。

**答案**：P ≈ 1/e ≈ 36.79%（N≥5 时已非常接近）。

**面试官考察**：容斥原理、与 e 的关系。

#### 题9：100层楼扔鸡蛋

**题目**：100层楼，2个鸡蛋，从某层扔下鸡蛋会碎（更高层也碎），求确定该临界层最少需要几次？

**思路**：设最优策略最多试 k 次，则第一个蛋应在第 k 层扔——若碎了还剩 k-1 次用第二个蛋线性扫。

**解答**：设最多试 k 次。第一个蛋依次从第 k, k+(k-1), k+(k-1)+(k-2), ... 层扔。能覆盖的楼层数 = k + (k-1) + ... + 1 = k(k+1)/2 ≥ 100。解得 k=14（因为 14×15/2 = 105 ≥ 100）。

**答案**：最少需要 14 次。

**面试官考察**：动态规划思维、最优化能力、能否用不等式快速估算。

#### 题10：随机选数配对问题

**题目**：N个人各从[1,100]中独立均匀随机选一个整数，选到相同数字的人配成一对，求期望配对数。

**思路**：期望线性性——对每个数字分别计算产生配对的期望贡献。

**解答**：对于某个特定数字 d，每人选中 d 的概率为 1/100。选中 d 的人数 X_d ~ Bin(N, 1/100)。该数字产生的配对数为 C(X_d, 2)。E[C(X_d, 2)] = C(N, 2) · (1/100)^2。100 个数字总期望配对数 = 100 · C(N, 2) / 10000 = C(N, 2) / 100。

**答案**：E = N(N-1)/200。例如 N=100 时，期望约 49.5 对。

**面试官考察**：期望线性性、指示变量法。

#### 题11：连续抽牌直到Ace出现

**题目**：从一副52张标准扑克牌中不放回地抽牌，直到第一张Ace出现为止，期望抽多少张牌？

**思路**：将4张Ace视为把48张非Ace牌分成5段，利用对称性。

**解答**：4张Ace把48张非Ace牌分成5个间隔（Ace之前、两两Ace之间、Ace之后）。由对称性，每个间隔的期望长度 = 48/5。第一张Ace之前的间隔期望长度 = 48/5 = 9.6，加上那张Ace本身，期望抽牌数 = 48/5 + 1 = 53/5 = 10.6。

**答案**：E = 53/5 = 10.6 张。

**面试官考察**：对称性论证、避免复杂计算的直觉。

#### 题12：条件概率陷阱——两个孩子问题

**题目**：一个家庭有两个孩子，已知至少一个是男孩，求两个都是男孩的概率。

**思路**：关键在于"已知至少一个是男孩"与"已知某个特定孩子是男孩"不同。

**解答**：两个孩子的性别组合：{BB, BG, GB, GG}，各概率1/4。"至少一个男孩"排除 GG，剩余 {BB, BG, GB}，在剩余中 BB 的条件概率 = (1/4)/(3/4) = 1/3。若题目改为"已知老大是男孩"，则答案变为 1/2——这是常见的条件概率陷阱。

**答案**：P = 1/3

**面试官考察**：条件概率、能否区分不同的信息条件。

#### 题13：Coupon Collector 问题

**题目**：有N种不同的优惠券，每次等概率随机获得一种，集齐所有N种的期望次数？

**思路**：分阶段——已有 k 种时，获得新种类的概率为 (N-k)/N，该阶段期望次数为 N/(N-k)。

**解答**：E = Σ_{k=0}^{N-1} N/(N-k) = N · Σ_{j=1}^{N} 1/j = N · H(N)，其中 H(N) 为第N个调和数。H(N) ≈ ln(N) + γ（γ ≈ 0.5772 为 Euler-Mascheroni 常数）。

**答案**：E = N·H(N) ≈ N·ln(N)。例如 N=50 时 E ≈ 225。

**面试官考察**：几何分布的期望、调和级数。

#### 题14：N个 U(0,1) 随机变量的最大值的期望

**题目**：X1, X2, ..., Xn 独立同分布于 U(0,1)，求 E[max(X1,...,Xn)]。

**思路**：先求 max 的 CDF，再求期望。

**解答**：设 M = max(X1,...,Xn)。P(M ≤ x) = P(所有 Xi ≤ x) = x^n（对 0≤x≤1）。E[M] = ∫₀¹ (1 - F(x)) dx = ∫₀¹ (1 - x^n) dx = 1 - 1/(n+1) = n/(n+1)。

**答案**：E[max] = n/(n+1)。如 n=10 时 E ≈ 0.909。

**面试官考察**：顺序统计量、CDF 求期望的技巧。

#### 题15：负二项分布

**题目**：抛一枚成功概率为 p 的硬币，直到第 k 次成功为止，求期望总试验次数。

**思路**：每次成功之间的等待时间服从几何分布，k 次成功的等待时间之和。

**解答**：第 i 次成功（i=1,...,k）到第 i+1 次成功之间的间隔（含成功那次）服从 Geom(p)，期望为 1/p。由期望线性性，总期望 = k · (1/p) = k/p。

**答案**：E = k/p

**面试官考察**：几何分布基础、期望线性性。

#### 题16：布丰投针问题

**题目**：平面上有间距为 d 的平行线，投长 l（l≤d）的针，求针与线相交的概率。

**思路**：针的位置由中心到最近线的距离 x 和针与线的夹角 θ 决定。

**解答**：x ~ U(0, d/2)，θ ~ U(0, π)。针与线相交条件：x ≤ (l/2)sinθ。P = (1/(d/2·π)) ∫₀^π ∫₀^{(l/2)sinθ} dx dθ = (1/(πd/2)) ∫₀^π (l/2)sinθ dθ = (l/(πd/2))·[−cosθ]₀^π·(1/2) = 2l/(πd)。

**答案**：P = 2l/(πd)。当 l=d 时 P = 2/π ≈ 0.6366。可用此估计 π。

**面试官考察**：几何概率、积分计算。

#### 题17：绳子切两刀组成三角形

**题目**：长为1的绳子随机切两刀分成三段，这三段能组成三角形的概率？

**思路**：三角形不等式要求每段 < 1/2。

**解答**：两个切点 X, Y ~ U(0,1) 独立。三段长度为排序后的间隔。在单位正方形中，满足所有三段 < 1/2 的区域面积就是所求概率。通过对称性和几何分析，满足条件的区域是正方形中心面积 = 1/4。

**答案**：P = 1/4 = 25%

**面试官考察**：几何概率、三角形不等式。

#### 题18：100个囚犯和灯泡问题

**题目**：100个囚犯，一间有灯泡（初始关）的房间。每天随机一人进入可开关灯。某人可以宣称"所有人都进过"。策略？

**思路**：选一个"计数员"，其他人只负责第一次开灯，计数员负责关灯计数。

**解答**：预先指定一名计数员。规则：(1) 非计数员第一次看到灯灭时开灯，之后不再操作；(2) 计数员看到灯亮就关灯并计数+1。当计数员关灯99次时，宣称所有人都已进过。期望时间约 10000 天（≈27年），但保证正确。

**答案**：计数员策略。计数员关灯99次后宣布，保证正确。

**面试官考察**：分布式协调、确定性算法 vs 概率算法的取舍。

#### 题19：期望的线性性——信封问题

**题目**：N封信随机放入N个信封，期望有几封信放对了信封？

**思路**：定义指示变量 Xi = 1（第i封信放对），E[Xi] = 1/N。

**解答**：设 Xi 为第 i 封信放对的指示变量。E[Xi] = P(第 i 封放对) = 1/N。期望放对总数 = E[ΣXi] = Σ E[Xi] = N · (1/N) = 1。注意：虽然 Xi 之间不独立，但期望的线性性不需要独立性。

**答案**：E = 1（与 N 无关）。

**面试官考察**：期望线性性的核心应用、指示变量法。

#### 题20：Secretary Problem（最优停止问题）

**题目**：面试 N 个候选人，逐一面试且当场决定录用或拒绝（不可回头），目标是选到最优候选人的概率最大。

**思路**：先看 N/e 个人（只看不选），之后选第一个比前面所有人都好的。

**解答**：最优策略——前 r 个人只观察不选择，从第 r+1 个人开始，选第一个比前 r 个人中最好的还好的人。最优的 r ≈ N/e。此时选到最优候选人的概率趋近于 1/e ≈ 36.8%。这是所有策略中能达到的最高概率，且当 N→∞ 时收敛到 1/e。

**答案**：最优策略为"观察前 N/e 人，然后选第一个超越的"，成功概率 ≈ 1/e ≈ 36.8%。

**面试官考察**：最优停止理论、1/e 法则、能否解释策略的直觉。

## Part 2：10道 Fermi 估算题

#### 估算1：中国A股市场每天的股票交易量（金额）

**思路分解**：总成交额 = 上交所 + 深交所日均成交额。

**估算过程**：上交所日均约 4000-5000 亿元，深交所约 5000-6000 亿元。两市合计日均约 8000 亿-1.2 万亿元。活跃行情时可达 1.5 万亿以上，低迷时约 6000 亿。

**答案**：量级约 1 万亿元人民币/天。

---

#### 估算2：一个基金经理每年看多少份研报

**思路分解**：每天工作时间 × 分配给研报的比例 ÷ 每份研报阅读时间。

**估算过程**：每天工作 10 小时，约 2-3 小时用于阅读研报。每份研报精读 30 分钟，速览 10 分钟，平均 15 分钟。每天约 10-12 份。一年 250 个交易日。250 × 10 ≈ 2500 份。

**答案**：量级约 2000-3000 份/年。

---

#### 估算3：上海有多少家量化私募

**思路分解**：全国私募总数 → 量化占比 → 上海占比。

**估算过程**：中国基金业协会登记的私募基金管理人约 2 万家，其中证券类约 8000 家。量化策略占证券私募约 15-20%，约 1200-1600 家。上海是量化私募聚集地，占全国约 30%。1400 × 30% ≈ 420 家。

**答案**：量级约 400-500 家。

---

#### 估算4：一天产生多少 tick 数据

**思路分解**：股票数量 × 每只股票每天 tick 数 + 期货等其他品种。

**估算过程**：A 股约 5000 只股票，每只交易时间 4 小时 = 14400 秒。Level-1 每 3 秒一个 snapshot，每只约 4800 条。5000 × 4800 = 2400 万条。Level-2 逐笔成交每只每天约 5-10 万笔，5000 × 7 万 = 3.5 亿条。加上期货约 100 个活跃合约 × 每日数万条 ≈ 数百万条。

**答案**：Level-1 量级约 2000-3000 万条；Level-2 逐笔约 3-5 亿条/天。

---

#### 估算5：全球外汇市场日交易量

**思路分解**：BIS 三年一次调查数据为基准进行估算。

**估算过程**：全球外汇市场是最大的金融市场，24 小时运转。主要交易中心：伦敦（约 40%）、纽约（约 15%）、东京（约 5%）、新加坡（约 8%）。BIS 2022 年调查日均约 7.5 万亿美元，近年增长约 5-10%/年。

**答案**：量级约 7-8 万亿美元/天。

---

#### 估算6：一个中频量化策略每天的交易笔数

**思路分解**：持仓股票数 × 每只股票日均交易次数。

**估算过程**：中频策略（持仓周期1-5天）通常持仓 50-200 只股票。每天换仓比例约 20-50%。假设持仓 100 只，每天换仓 30 只（买卖各 30 笔 = 60 笔）。加上日内调仓、拆单等，每笔大单可能拆成 10-50 个子单。60 × 20 = 1200 笔子单。

**答案**：量级约 1000-2000 笔/天（含拆单）。

---

#### 估算7：中国有多少个人股票投资者

**思路分解**：开户数 × 活跃比例。

**估算过程**：中国证券开户数约 2-2.2 亿个（自然人）。但活跃投资者（过去一年有交易）约 5000-7000 万。经常交易的（每月至少一次）约 2000-3000 万。

**答案**：开户量级约 2 亿，活跃投资者约 5000 万-7000 万。

---

#### 估算8：一个量化团队每年的算力成本

**思路分解**：GPU 服务器 + CPU 回测集群 + 数据存储 + 网络。

**估算过程**：中型量化团队（20-50 人）。GPU 服务器 4-8 台 × 30 万/年 ≈ 150 万。CPU 集群约 50 台 × 5 万/年 ≈ 250 万。数据费用（万得/聚宽/交易所直连）约 100-200 万。行情专线约 50 万。云服务约 100 万。

**答案**：量级约 500 万-1000 万元/年。

---

#### 估算9：全球对冲基金管理的总资产规模

**思路分解**：头部基金规模 + 长尾分布估算。

#### 估算10：沪深300指数一天的波动率是多少

## Part 3：5道金融直觉题

#### 直觉1：Black-Scholes 公式的直觉含义

不写公式，纯文字理解 C = S·N(d1) - K·e^{-rT}·N(d2)：

- **S·N(d1)**：可以理解为"在期权到期时为实值的条件下，你将获得的股票价值的期望"。N(d1) 是对冲比率(Delta)，代表股价变动1元时期权价格的变动量。它也是期权到期时为实值的风险中性概率经过一次调整后的值。

- **K·e^{-rT}·N(d2)**：这是你到期时需要支付的行权价的现值，乘以"期权到期时为实值的风险中性概率" N(d2)。换言之，这是你"期望支付的行权价的折现值"。

- **整体含义**：期权价值 = 期望收到的股票价值 - 期望支付的行权价现值。这与直觉一致：Call 的价值就是"期望能拿到的东西"减去"期望要付出的东西"。

- **d1 和 d2 的区别**：d2 = d1 - σ√T。d2 对应的是在风险中性测度下期权到期为实值的概率；d1 对应的是在股票价格测度（以股票为计价单位）下的概率。两者相差一个波动率调整项。

---

#### 直觉2：Greeks 的直觉理解

- **Delta (Δ)**：期权价格对标的资产价格的敏感度。Call 的 Delta 在 0 到 1 之间。深度实值 Delta→1（期权像股票），深度虚值 Delta→0（期权几乎不反应）。也可以近似理解为期权到期时为实值的概率。

- **Gamma (Γ)**：Delta 的变化速率，即期权价格对标的价格的二阶导。Gamma 在平值附近最大，说明平值期权的 Delta 对股价最敏感。Gamma 是"凸性收益"的来源——做多 Gamma 意味着股票涨时 Delta 变大（赚更多），跌时 Delta 变小（亏更少）。

- **Theta (Θ)**：时间流逝对期权价值的侵蚀。几乎总是负值（期权每天在贬值）。Theta 在平值、临近到期时最大。Theta 与 Gamma 是一对矛盾——做多 Gamma（享受凸性）就要付出 Theta（时间损耗），这就是期权市场的核心 trade-off。

- **Vega (ν)**：期权价格对隐含波动率的敏感度。长期限、平值期权 Vega 最大。做多 Vega 就是赌波动率上升。在实践中，交易员最关注的往往是 Vega 而非 Delta——因为波动率的不确定性比方向更难对冲。

---

#### 直觉3：Put-Call Parity 的直觉证明

**结论**：C - P = S - K·e^{-rT}

**直觉证明**：构造两个组合，证明它们到期时的 payoff 完全一样：

- **组合 A**：买一个 Call + 持有 K·e^{-rT} 的现金（到期变成 K）
- **组合 B**：买一个 Put + 持有一股股票

到期时：若 S_T > K，组合 A 的 Call 行权得 S_T，总值 S_T；组合 B 的 Put 作废，总值 S_T。若 S_T < K，组合 A 的 Call 作废，总值 K；组合 B 的 Put 行权得 K，总值 K。

两个组合到期 payoff 完全相同，由无套利原则，现在的价值也必须相同：C + K·e^{-rT} = P + S，即 C - P = S - K·e^{-rT}。

**实际用途**：用 Put-Call Parity 可以从 Call 价格推 Put 价格（反之亦然），也可以检测市场是否存在套利机会。

#### 直觉4：为什么波动率微笑存在

Black-Scholes 假设波动率为常数，但实际市场中不同行权价的期权隐含波动率不同，画出来呈"微笑"或"偏斜"形状。原因：

1. **尾部风险被低估**：BS 模型假设对数正态分布，但现实中股价的尾部（极端涨跌）比正态分布更厚（肥尾）。深度虚值期权（尤其是 Put）因此被市场定价更高，反映出更高的隐含波动率。

2. **跳跃风险**：股价可能出现突然跳跃（如闪崩），这在 BS 的连续扩散模型中不存在。市场通过提高虚值 Put 的隐含波动率来补偿这一风险。

3. **杠杆效应**：股价下跌时公司杠杆上升（债务/市值比增大），导致波动率上升。这使得低行权价对应更高的隐含波动率，形成"偏斜"而非对称微笑。

4. **供需因素**：机构投资者大量买入虚值 Put 作为保险（portfolio protection），推高了虚值 Put 的价格和隐含波动率。

5. **1987 年黑色星期一之后**：在此之前波动率曲面相对平坦。崩盘后市场对尾部风险的认知彻底改变，波动率微笑成为永久特征。

#### 直觉5：无风险套利 vs 统计套利的区别

| 维度 | 无风险套利 | 统计套利 |
|------|-----------|---------|
| **定义** | 零投入、零风险、正收益 | 基于统计规律的预期正收益 |
| **风险** | 理论上无风险 | 有风险（模型风险、执行风险） |
| **持续时间** | 瞬间或极短 | 需要持有一段时间 |
| **存在性** | 在有效市场中几乎不存在 | 广泛存在但收益有限 |
| **例子** | 三角外汇套利、ETF 折溢价套利 | 配对交易、均值回归策略 |
| **资金需求** | 理论上零（实际需要保证金） | 需要本金承担风险 |
| **确定性** | 确定盈利 | 大数定律下期望盈利 |

**关键区别**：无风险套利是"free lunch"——违反一价定律时出现，在发达市场中被高频交易者迅速消除，存活时间以毫秒计。统计套利是"lunch with a small price"——基于历史统计规律（如配对股票的价差会回归均值），但单次交易可能亏损，只有大量交易后才体现期望优势。

统计套利的风险来源：(1) 模型假设失效（regime change）；(2) 极端事件（如 LTCM 1998）；(3) 拥挤交易（同类策略太多导致alpha衰减）；(4) 执行滑点和交易成本。
