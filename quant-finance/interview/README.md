## 面试准备建议

### 每日练习计划（4 周冲刺）

**第 1 周：概率基础**
- 每天做 2-3 道概率题，写出完整推导
- 重点掌握：条件概率、期望的线性性、指示变量法
- 每道题限时 10 分钟，超时看答案后隔天重做

**第 2 周：策略与金融直觉**
- 每天精读 1 道策略设计题，理解答题框架
- 补充阅读：[[concepts/factor_investing_framework.md]]、[[concepts/risk_measures.md]]
- 练习口述答题（面试是讲出来的，不是写出来的）

**第 3 周：编程与实操**
- pandas 时序操作、回测框架搭建
- 每天 1 道 LeetCode（侧重数组/矩阵/概率模拟）
- 熟悉 numpy 向量化操作

**第 4 周：综合模拟**
- 每天模拟 1 场完整面试（45 分钟）
- 混合题型：概率 → 策略 → 编程
- 录音回放，改进表达

### 面试通用技巧

1. **先说思路再算**：面试官看的是思维过程，不只是答案
2. **边界情况**：概率题注意 0 和 1 的特殊情况
3. **量纲检查**：策略题注意收益率、波动率的时间尺度是否一致
4. **追问准备**：每个答案准备一个"如果面试官追问..."的延伸
