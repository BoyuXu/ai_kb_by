# Enhancing Generative Auto-Bidding with Prompt-Based Constraints
> 来源：工业论文 | 领域：广告系统 | 学习日期：20260327

## 问题定义
LLM应用于自动出价（AutoBidding）面临约束满足挑战：
1. **预算约束**：广告主设定每日预算上限，系统必须严格遵守
2. **ROI约束**：广告主要求每元广告费至少带来X元回报
3. **目标曝光约束**：品牌广告主要求至少达到X次曝光
LLM直接生成出价时往往忽略这些约束，导致超预算或未达目标。

**目标**：通过Prompt工程和约束感知训练，让LLM生成的出价严格满足多维度约束，约束满足率+25%。

## 核心方法与创新点

### 1. 约束感知Prompt设计
将约束显式编码进Prompt：
```
当前状态：
- 今日已花费：¥456（预算¥1000，剩余¥544）
- 当前ROI：3.2（目标ROI≥3.0，✅达标）
- 今日曝光：125,000（目标≥200,000，⚠️需加速）

历史出价统计：上午均价¥0.8，下午均价¥1.2

指令：在满足ROI≥3.0的约束下，根据当前剩余预算和曝光缺口，给出本次最优出价。
输出格式：{"bid": <金额>, "reason": "<出价理由>"}
```

### 2. 约束感知微调
用历史出价数据+约束满足情况构建训练集：
- 正样本：约束满足的出价决策（预算不超、ROI达标）
- 负样本：违反约束的出价决策
用DPO（Direct Preference Optimization）对比训练。

### 3. 约束投影层
LLM生成出价后，通过约束投影层进行后处理：
$$
b_{final} = \text{clip}(b_{LLM}, b_{min}, b_{max}(\text{remaining\_budget}, \text{ROI\_constraint}))
$$

$b_{max}$ 由剩余预算和ROI约束动态计算：
$$
b_{max} = \min(\text{remaining\_budget}/\text{remaining\_slots}, \text{expected\_value}/\text{ROI\_target})
$$

### 4. 多步推理（Chain-of-Thought）
让LLM显式进行约束推理后再给出出价：
```
思考步骤：
1. 当前剩余预算/剩余时间 = 约束出价上限
2. 预期转化价值/ROI目标 = 价值出价上限
3. min(1, 2) = 约束出价
4. 结合竞争市场价格调整
→ 最终出价
```

## 实验结论
- **约束满足率**：+25%（综合预算、ROI、曝光三维约束）
- **预算利用率**：95%（在满足ROI约束前提下，预算尽量花完）
- **ROI达成率**：92%（相比规则方案的78%）
- CoT出价推理比直接出价约束满足率高+10%

## 工程落地要点
1. **Prompt实时更新**：约束状态（已花费金额、当前ROI）需要实时查询，每次出价前刷新
2. **推理延迟**：CoT推理比直接出价慢5-10ms，需要在延迟预算内权衡推理深度
3. **约束优先级**：明确约束优先级（预算硬约束 > ROI软约束 > 曝光目标），避免约束冲突
4. **异常处理**：LLM输出格式异常时，有规则兜底方案（回退到简单出价策略）
5. **广告主教育**：设置过于严苛约束（ROI>100）会让系统无法出价，需要在界面给出合理约束建议

## 面试考点
Q1: 为什么约束满足对广告主如此重要？
A: 广告主的预算和ROI约束直接关系到商业利益：预算超支 = 亏损，ROI不达标 = 广告投放不划算。传统基于规则的出价调整逻辑简单（如"预算快用完了就降价"），无法同时优化多个相互关联的约束。LLM能够综合考虑多维度状态，在满足约束的前提下寻找全局最优出价策略。

Q2: DPO（直接偏好优化）相比PPO在约束学习中有什么优势？
A: PPO：需要单独训练奖励模型，再用RL优化策略，实现复杂，容易不稳定。DPO：直接从偏好数据（约束满足vs违反）对比优化语言模型，无需独立奖励模型，训练更稳定，实现更简单。对于约束满足这类有明确偏好定义的场景，DPO的效果与PPO相当甚至更好。

Q3: CoT（思维链）出价推理的风险是什么？
A: (1)推理错误放大：CoT的中间步骤如果出错，会传播到最终出价决策，比直接出价的错误更系统性；(2)延迟增加：每次出价需要生成数十个中间token，RTB系统延迟预算可能不够；(3)过度解释：LLM可能生成听起来合理但实际上不优的推理链，对关键参数（剩余预算）计算出错。建议：CoT推理用于确认约束检查，最终出价仍用约束投影层保证安全。
