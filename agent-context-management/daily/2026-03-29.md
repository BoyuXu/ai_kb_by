## 💡 面试相关知识点提炼

**Q: LLM Agent 的 memory 有哪几种类型？**
- **In-context memory（工作记忆）**：当前 context window 内的信息，容量受 token 限制
- **External memory（外部记忆）**：向量数据库、知识图谱（GSEM），支持长期存储与检索
- **Persistent memory（持久化记忆）**：跨会话保存（Memori、MemArchitect），需要治理层
- **Parametric memory（参数记忆）**：模型权重本身存储的知识，通过 fine-tuning 更新

**Q: 如何扩展 LLM 的有效上下文长度？**
- 稀疏注意力（MSA - Memory Sparse Attention）
- 滑动窗口 + 压缩摘要
- 分层/树状上下文结构（Conversation Tree）
- 外部检索增强（RAG）

**Q: 多 Agent 系统的 memory 共享如何实现？**
- MemCollab：对比轨迹蒸馏 → 跨 agent 记忆迁移
- 共享外部记忆库（向量 DB）
- Memory poisoning 安全问题需要治理层（MemArchitect）
