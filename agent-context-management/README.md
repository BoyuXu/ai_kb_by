## 🗺️ 核心主题地图

### 记忆类型分层
```
┌─────────────────────────────────────────────┐
│  外部记忆（External Memory）                │
│  ┌─────────────┬────────────────────────┐   │
│  │ 向量数据库  │  结构化 KV / 图数据库  │   │
│  └─────────────┴────────────────────────┘   │
├─────────────────────────────────────────────┤
│  上下文记忆（Context Memory）               │
│  ┌──────────────────────────────────────┐   │
│  │  KV Cache  │  Sparse Attention       │   │
│  └──────────────────────────────────────┘   │
├─────────────────────────────────────────────┤
│  工作记忆（Working Memory）                 │
│  ┌──────────────────────────────────────┐   │
│  │  Context Window (当前 Prompt)        │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### 关键技术线索
- **如何存储？** → Memori / MemArchitect（持久化 + 治理）
- **如何扩展？** → MSA / KV Cache Opt（亿级 token）
- **如何组织？** → Conversation Tree / Library Theorem（结构化）
- **如何共享？** → MemCollab（跨 Agent 协作）
- **如何评测？** → VehicleMemBench / BeliefShift
