├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │

**P5 优缺点:**

| 优点 | 缺点 |
|-----|-----|
| 任务统一，减少多模型维护 | 推理成本高，延迟大 |
| 零样本/少样本能力强 | 商品数量受限(T5词汇表) |
| 自然可解释性 | 冷启动仍需fine-tuning |
| 跨领域迁移潜力 | 长序列处理受限 |

### 1.5 LLM-as-RecModel 对比总结

| 方法 | 核心思想 | 模型 | 训练成本 | 推理延迟 | 适用场景 |
|-----|---------|------|---------|---------|---------|
| **P5** | 统一文本生成 | T5-base | 中 | 高 | 多任务统一、研究 |

### 2.3 特征增强实践代码

```python
# LLM特征增强示例
from transformers import AutoTokenizer, AutoModel
import torch.nn.functional as F

class LLMFeatureEnhancer:
    def __init__(self, model_name="sentence-transformers/all-MiniLM-L6-v2"):
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model = AutoModel.from_pretrained(model_name)
        self.model.eval()
    
    def encode_item(self, title, description, category):
        """编码商品为语义向量"""
        text = f"{title}. {description}. Category: {category}"
        
        inputs = self.tokenizer(
            text, 
            return_tensors="pt", 
            truncation=True, 
            max_length=512
        )
        
        with torch.no_grad():
            outputs = self.model(**inputs)
            # Mean pooling
            embeddings = self.mean_pooling(outputs, inputs['attention_mask'])
            embeddings = F.normalize(embeddings, p=2, dim=1)
        
        return embeddings.squeeze().numpy()
    
    def generate_user_profile(self, interaction_history):
        """
        基于交互历史生成用户画像文本
        可用于zero-shot推荐或作为特征
        """
        items_text = ", ".join([
            f"{item['title']}({item['category']})" 
            for item in interaction_history[-10:]
        ])
        
        prompt = f"""
        Based on the user's recent interactions: {items_text}
        
        Describe this user's preferences and interests in 2-3 sentences.
        Focus on: product categories, brands, price sensitivity, style preferences.
        """
        
        # 使用LLM生成画像描述
        profile = self.llm_generate(prompt)
        return profile
    
    def semantic_similarity_recall(self, query_item, candidate_pool, top_k=100):
        """基于语义相似度的召回 (解决新物品冷启动)"""
        query_vec = self.encode_item(**query_item)
        
        candidate_vecs = [
            self.encode_item(**item) for item in candidate_pool
        ]
        
        # 计算相似度
        similarities = cosine_similarity([query_vec], candidate_vecs)[0]
        
        # 返回top-k相似物品
        top_indices = np.argsort(similarities)[-top_k:][::-1]
        return [candidate_pool[i] for i in top_indices]
```

### 3.4 对话式推荐挑战

| 挑战 | 描述 | 解决方案 |
|-----|------|---------|
| **幻觉问题** | LLM推荐不存在的商品 | 检索增强(RAG)、工具调用验证 |
| **实时性** | LLM推理延迟高 | 流式输出、缓存、意图预分类 |
| **一致性** | 多轮推荐不一致 | 对话状态追踪、显式记忆管理 |
| **评估困难** | 缺乏对话推荐评估标准 | 模拟用户评估、A/B测试框架 |
| **安全性** | 不当推荐、隐私泄露 | 内容过滤、输出约束、联邦学习 |

### 4.3 检索策略

```python
# RAG推荐检索策略示例
class RAGRecommender:
    def __init__(self, vector_store, llm):
        self.vector_store = vector_store
        self.llm = llm
    
    def build_query(self, user_profile, current_intent, conversation_history):
        """
        构建检索Query
        融合用户画像、当前意图、对话上下文
        """
        query_parts = []
        
        # 1. 显式需求
        query_parts.append(current_intent)
        
        # 2. 隐式偏好 (从画像推断)
        implicit_prefs = self.infer_preferences(user_profile)
        query_parts.extend(implicit_prefs)
        
        # 3. 约束条件
        constraints = self.extract_constraints(conversation_history)
        
        # 4. 构建多样化Query
        queries = [
            f"{current_intent} {' '.join(implicit_prefs)}",  # 语义搜索
            self.structured_query(current_intent, constraints),  # 属性过滤
            self.expand_query(current_intent),  # 同义扩展
        ]
        
        return queries
    
    def multi_strategy_retrieve(self, queries, top_k=50):
        """多策略融合检索"""
        all_results = []
        
        # 1. 向量语义检索
        semantic_results = self.vector_store.similarity_search(
            queries[0], k=top_k, filter={"type": "item_description"}
        )
        all_results.extend([(r, "semantic", score) for r, score in semantic_results])
        
        # 2. 属性精确匹配
        attr_results = self.vector_store.filter_search(
            queries[1], k=top_k
        )
        all_results.extend([(r, "attribute", score) for r, score in attr_results])
        
        # 3. 个性化重排序 (使用用户画像)
        reranked = self.personalized_rerank(all_results, user_profile)
        
        # 4. 多样性增强 (MMR算法)
        diverse_results = self.maximal_marginal_relevance(reranked, lambda_param=0.5)
        
        return diverse_results[:20]
    
    def generate_with_rag(self, query, retrieved_items, user_context):
        """基于检索结果生成推荐"""
        
        context = self.build_context(retrieved_items)
        
        prompt = f"""Based on the following relevant product information:

{context}

User Context: {user_context}
User Query: {query}

Please recommend the top 5 most suitable products and explain why:

Requirements:
1. Consider both relevance to query and user preferences
2. Provide specific reasons for each recommendation
3. Mention any trade-offs if applicable
4. Format as JSON list
"""
        
        response = self.llm.generate(prompt)
        return self.parse_recommendations(response)
```

### 4.4 RAG+推荐优势总结

| 维度 | 传统推荐 | RAG+推荐 |
|-----|---------|---------|
| **知识更新** | 需重训练模型 | 实时更新知识库 |
| **长尾覆盖** | 数据稀疏难学习 | 检索补充知识 |
| **可解释性** | 黑盒模型 | 可追溯推荐来源 |
| **灵活性** | 固定模型能力 | 动态组合知识 |
| **冷启动** | 依赖历史数据 | 知识库直接支持 |
| **计算成本** | 推理一次 | 检索+生成两次 |
