# LLM-Enhanced Ad Creative Generation and Optimization for E-Commerce

> 来源：arxiv | 日期：20260321 | 领域：ads

## 问题定义

电商广告的创意（Ad Creative）包括标题、描述文案、主图等，直接影响用户点击率（CTR）和转化率（CVR）。传统创意制作面临：

1. **规模化瓶颈**：SKU 数百万，人工为每个商品定制创意不可行
2. **个性化不足**：同一广告面向不同用户群展示相同文案，CTR 损失显著
3. **迭代速度慢**：A/B 测试文案需要数天收集足够数据，市场热点错过

LLM 的强大文本生成能力为规模化个性化广告创意提供了可能，但直接生成的文案不一定符合平台规范和用户偏好，需要专门的优化框架。

## 核心方法与创新点

1. **电商场景微调 LLM**：
   - 在商品标题、卖点、用户评论、历史高 CTR 文案数据上微调 LLaMA/Qwen
   - 构建"商品属性 → 广告文案"的指令微调数据集
   - 引入平台规范过滤（违禁词、夸大宣传检测）作为训练数据清洗步骤

2. **CTR 感知的文案优化**：
   - 生成多个候选文案（温度采样，Beam Search Top-K）
   - 用轻量 CTR 预估模型对候选文案打分（将文案 embedding 作为特征）
   - 选择 CTR 预估分最高的文案投放

3. **用户群体感知生成（Persona-Conditioned）**：
   - 将用户群体特征（年龄段、消费能力、兴趣标签）作为 prompt 条件
   - 为"年轻女性" vs "中年男性"生成不同风格文案
   - 实现"千人千面"的创意个性化

4. **在线反馈闭环**：
   - 投放后收集真实 CTR/CVR 数据，自动识别高效文案
   - 将高效文案加入微调数据集，实现持续优化（Online Learning）

## 实验结论

- 相比人工编写文案：CTR **+8.2%**，CVR **+5.7%**，规模从覆盖 10% SKU 提升至 **100%**
- 个性化文案（Persona-Conditioned）相比通用文案：目标人群 CTR **+14.3%**
- 生成速度：10,000 条文案/分钟（GPU 批量推理），比人工快 1000 倍以上
- 内容安全合规率：98.7%（需保留人工审核 1.3%）

## 工程落地要点

1. **文案合规自动化**：必须部署违禁词过滤 + 夸大宣传检测（基于规则+分类器），LLM 偶尔会生成合规性问题文案，不可直接投放
2. **多版本 A/B 测试**：同一商品同时投放 2-3 个 LLM 生成变体，快速收敛到最优版本（Bandit 算法加速）
3. **延迟与质量平衡**：在线请求生成需要 <500ms，离线批量生成可接受数分钟；工业上对实时场景用预生成缓存
4. **版权与品牌一致性**：生成文案需检查是否与品牌调性一致，高端品牌不能用廉价感文案，需要品牌风格约束 prompt

## 常见考点

- Q: 广告文案生成如何做质量评估？
  A: (1) 自动指标：ROUGE/BERTScore（与高质量参考文案相似度）；(2) 分类器评估：CTR 预估模型打分；(3) 人工评估（合规性、相关性、吸引力）；(4) 最终指标：线上 A/B 测试的真实 CTR/CVR。自动指标和线上效果相关性有限，最终还是要靠 A/B 测试。

- Q: LLM 生成广告文案的主要风险有哪些？
  A: (1) 合规风险：夸大宣传（"最便宜"/"第一"等绝对词）、违禁词；(2) 事实错误：LLM 可能捏造商品不存在的属性；(3) 品牌风险：文案风格与品牌不符；(4) 版权风险：可能复制竞品文案。每个风险都需要对应的检测和过滤机制。

- Q: 如何让 LLM 生成的文案更符合平台风格？
  A: (1) 在平台历史高效文案上微调（SFT）；(2) RLHF：用 CTR/CVR 作为奖励信号做强化微调；(3) Prompt Engineering：在 prompt 中提供平台风格示例（Few-shot）；(4) RAG：检索相似商品的历史高效文案作为参考。

## 模型架构详解

### 内容生成管线
- **输入理解**：广告主目标（品牌/效果）、商品属性、目标人群画像
- **创意生成**：LLM/扩散模型生成文案/图片候选
- **质量评估**：CTR 预估模型打分 + 品牌安全检测 + 多样性筛选

### 多模态融合
- 文本编码：BERT/LLM 提取语义特征
- 图像编码：ViT/CLIP 提取视觉特征
- 跨模态对齐：对比学习确保文图一致性

### 在线服务架构
- 候选池管理：预生成创意库 + 实时个性化选择
- A/B 测试框架：多臂老虎机选择最优创意变体
- 反馈闭环：展示数据回流更新生成模型

## 与相关工作对比

| 维度 | 本文方法 | 传统方法 | 优势 |
|------|---------|---------|------|
| 创意生成 | LLM/生成模型 | 模板+人工 | 规模化个性化 |
| 质量控制 | 自动评估+人工审核 | 纯人工 | 效率提升10x |
| 个性化 | 用户级定制 | 人群级 | 更精准匹配 |

## 面试深度追问

- **Q: 如何评估生成创意的质量？**
  A: 多维度评估：1) CTR 预估模型打分；2) 品牌安全检测（敏感词/不当内容）；3) 多样性评估（避免同质化）；4) 人工审核采样。

- **Q: LLM 生成广告文案的幻觉问题如何解决？**
  A: 1) RAG 检索商品属性作为 grounding；2) 后处理校验（价格/规格/品牌名一致性检查）；3) Few-shot prompt 约束输出格式。

- **Q: 创意千人千面的技术方案？**
  A: 1) 用户兴趣标签驱动模板选择；2) 动态创意组合（DCO）：标题 × 图片 × CTA 按钮的组合优化；3) 基于 Contextual Bandit 的在线创意选择。

- **Q: 图片生成质量不稳定怎么办？**
  A: 1) 预生成 + 人工审核建立高质量创意库；2) CLIP Score 自动过滤低质量生成；3) ControlNet 约束生成风格一致性。
