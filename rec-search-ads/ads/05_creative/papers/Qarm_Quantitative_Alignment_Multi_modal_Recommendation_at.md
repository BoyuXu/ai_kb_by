# Qarm: Quantitative Alignment Multi-modal Recommendation at Kuaishou
> 来源：https://arxiv.org/abs/2411.11739 | 领域：ads | 日期：20260323

## 问题定义
快手广告推荐系统中，多模态特征（视频、音频、文本、用户行为）的对齐与融合是关键挑战。Qarm提出量化对齐的多模态推荐框架，实现视频广告的精准推荐。

## 核心方法与创新点
- 量化对齐：将不同模态的连续特征量化为离散token，统一表示空间
- 跨模态注意力：视频帧、音频、字幕之间的细粒度交互
- 行为增强：用用户交互行为监督多模态表示的学习方向
- 工业级效率：针对快手亿级视频的特征提取和索引优化

## 实验结论
快手广告系统实验：Qarm相比单模态基线，CTR提升1.3%，视频完播率提升0.8%；多模态对齐相比简单拼接提升约0.5%；量化表示相比连续embedding推理速度提升3x。

## 工程落地要点
- 视频特征提取需要GPU集群，通常离线批量处理（每视频约1-10秒）
- 量化码本需要支持动态扩展，适应新视频内容的语义分布变化
- 短视频广告的音视频同步对齐是工程难点，需要精确的帧-字幕对齐

## 常见考点
1. **Q: 短视频广告推荐的特殊挑战？** A: 视频内容理解复杂（多模态）、用户停留时间短（几秒决策）、时效性强（爆款视频快速变化）
2. **Q: 多模态融合的常见方案？** A: 早期融合（特征拼接）、晚期融合（预测融合）、交叉融合（交叉注意力）
3. **Q: 量化表示为何能提升推理速度？** A: 向量检索变为离散码本查找，可用整数运算代替浮点计算，降低内存带宽
4. **Q: 视频广告的质量评估维度？** A: 内容相关性、用户体验（非侵入性）、品牌安全（不出现负面内容旁边）
5. **Q: 快手多模态系统的数据规模挑战？** A: 每天新增千万级视频，特征提取需要分布式流水线，无法全量实时处理

## 模型架构详解

### 内容生成管线
- **输入理解**：广告主目标（品牌/效果）、商品属性、目标人群画像
- **创意生成**：LLM/扩散模型生成文案/图片候选
- **质量评估**：CTR 预估模型打分 + 品牌安全检测 + 多样性筛选

### 多模态融合
- 文本编码：BERT/LLM 提取语义特征
- 图像编码：ViT/CLIP 提取视觉特征
- 跨模态对齐：对比学习确保文图一致性

### 在线服务架构
- 候选池管理：预生成创意库 + 实时个性化选择
- A/B 测试框架：多臂老虎机选择最优创意变体
- 反馈闭环：展示数据回流更新生成模型

## 面试深度追问

- **Q: 如何评估生成创意的质量？**
  A: 多维度评估：1) CTR 预估模型打分；2) 品牌安全检测（敏感词/不当内容）；3) 多样性评估（避免同质化）；4) 人工审核采样。

- **Q: LLM 生成广告文案的幻觉问题如何解决？**
  A: 1) RAG 检索商品属性作为 grounding；2) 后处理校验（价格/规格/品牌名一致性检查）；3) Few-shot prompt 约束输出格式。

- **Q: 创意千人千面的技术方案？**
  A: 1) 用户兴趣标签驱动模板选择；2) 动态创意组合（DCO）：标题 × 图片 × CTA 按钮的组合优化；3) 基于 Contextual Bandit 的在线创意选择。

- **Q: 图片生成质量不稳定怎么办？**
  A: 1) 预生成 + 人工审核建立高质量创意库；2) CLIP Score 自动过滤低质量生成；3) ControlNet 约束生成风格一致性。

- **字节跳动**：巨量引擎中的智能创意（自动生成广告标题/封面）

- **Meta**：Advantage+ Creative 自动优化创意元素

### 关键指标参考
- 创意生成效率：单条广告 10+ 创意变体 / 秒
- CTR 提升：智能创意 vs 人工创意通常 5%~15% CTR 提升
- 人工审核通过率：生成创意的品牌安全通过率 > 95%

### 部署注意事项
1. 内容安全：必须经过多层过滤（敏感词 + 图像安全 + 人工抽检）
2. 版权合规：生成内容不能侵犯商标/肖像权
3. A/B 灰度：新创意策略需小流量验证后逐步放量
