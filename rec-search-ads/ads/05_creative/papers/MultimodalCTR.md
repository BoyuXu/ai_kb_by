# WWW 2025 多模态 CTR 预测挑战赛冠军方案

> 来源：1st Place Solution of WWW 2025 EReL@MIR Workshop Multimodal CTR Prediction Challenge | 日期：20260318 | 领域：ads

## 问题定义

多模态 CTR 预测：在传统用户行为特征（ID 特征、统计特征）的基础上，融合广告创意的图像、视频、文本等多模态内容特征，预测用户点击率。挑战在于：
1. 多模态特征维度高、模态间语义鸿沟大。
2. 视频/图像特征提取计算成本高，与在线系统延迟要求矛盾。
3. 多模态特征与用户行为 ID 特征的融合策略设计。

## 核心方法与创新点

冠军方案采用多层次融合 + 集成策略：

1. **多模态特征提取**：
   - 图像：ViT-L/CLIP 提取语义 embedding；ResNet 提取视觉风格特征。
   - 视频：逐帧采样后做时序平均池化；关键帧 CLIP embedding。
   - 文本：标题/描述用 BGE/E5 编码；类目用 category embedding。

2. **跨模态对齐**：用对比学习（CLIP 风格）对齐图像和文本 embedding，使同一广告的图文在 embedding 空间靠近，减少模态鸿沟。

3. **行为序列 × 多模态特征融合**：
   - 用户历史点击广告的多模态 embedding 做序列平均，作为"多模态兴趣画像"。
   - Target Attention：用当前广告多模态特征作为 query，对用户历史做 attention 加权。

4. **集成策略**：训练多个模型（DeepFM + MMOE + DCNv2），各模型用不同多模态特征组合，最终 Stacking 集成；对每个模态分别做消融选择最优特征子集。

## 实验结论

- 赛题 AUC 最终 0.8312，较基线（纯 ID 特征）提升约 0.012。
- 多模态特征贡献约 0.007 AUC，集成再贡献约 0.005 AUC。
- 图像特征 > 视频特征 > 文本特征（该数据集上的相对贡献）。
- 跨模态对齐使 AUC 额外提升约 0.002。

## 工程落地要点

- **离线预计算**：CLIP/ViT embedding 全量离线计算并存入向量数据库（Milvus/Faiss），在线检索延迟 < 5ms。
- **特征维度压缩**：ViT-L 输出 768 维，线上特征需 PCA 压缩到 64-128 维，减少存储和计算开销。
- **视频特征的代理**：视频全量编码成本极高，工业中常用封面图代替视频特征，只对 top 视频做帧级编码。
- **训练数据构建**：多模态 CTR 数据需保证正负样本中创意素材有效（非占位符图），否则引入噪声。

## 常见考点

**Q: 多模态特征如何与 ID 特征融合？**
- 早融合（early fusion）：特征拼接后输入模型；晚融合（late fusion）：各模态独立预测后集成；注意力融合：以一种模态为 query 对其他模态做 attention；工业常用 late + attention 结合。

**Q: CLIP embedding 为什么适合广告 CTR 场景？**
- CLIP 在大规模图文对上预训练，语义对齐好；广告创意通常有图文配合，CLIP 能捕捉两者语义一致性；zero-shot 泛化能力强，对新广告主的新素材无需重新训练。

**Q: 多模态 CTR 模型的训练数据偏差问题？**
- 展示偏差（只有曝光的广告有点击标签）；素材质量偏差（优质素材被选中展示的频率高）；需要在训练时做 IPW 去偏。

## 模型架构详解

### 内容生成管线
- **输入理解**：广告主目标（品牌/效果）、商品属性、目标人群画像
- **创意生成**：LLM/扩散模型生成文案/图片候选
- **质量评估**：CTR 预估模型打分 + 品牌安全检测 + 多样性筛选

### 多模态融合
- 文本编码：BERT/LLM 提取语义特征
- 图像编码：ViT/CLIP 提取视觉特征
- 跨模态对齐：对比学习确保文图一致性

### 在线服务架构
- 候选池管理：预生成创意库 + 实时个性化选择
- A/B 测试框架：多臂老虎机选择最优创意变体
- 反馈闭环：展示数据回流更新生成模型

## 面试深度追问

- **Q: 如何评估生成创意的质量？**
  A: 多维度评估：1) CTR 预估模型打分；2) 品牌安全检测（敏感词/不当内容）；3) 多样性评估（避免同质化）；4) 人工审核采样。

- **Q: LLM 生成广告文案的幻觉问题如何解决？**
  A: 1) RAG 检索商品属性作为 grounding；2) 后处理校验（价格/规格/品牌名一致性检查）；3) Few-shot prompt 约束输出格式。

- **Q: 创意千人千面的技术方案？**
  A: 1) 用户兴趣标签驱动模板选择；2) 动态创意组合（DCO）：标题 × 图片 × CTA 按钮的组合优化；3) 基于 Contextual Bandit 的在线创意选择。

- **Q: 图片生成质量不稳定怎么办？**
  A: 1) 预生成 + 人工审核建立高质量创意库；2) CLIP Score 自动过滤低质量生成；3) ControlNet 约束生成风格一致性。

### 关键指标参考
- 创意生成效率：单条广告 10+ 创意变体 / 秒
- CTR 提升：智能创意 vs 人工创意通常 5%~15% CTR 提升
- 人工审核通过率：生成创意的品牌安全通过率 > 95%

### 部署注意事项
1. 内容安全：必须经过多层过滤（敏感词 + 图像安全 + 人工抽检）
2. 版权合规：生成内容不能侵犯商标/肖像权
3. A/B 灰度：新创意策略需小流量验证后逐步放量
