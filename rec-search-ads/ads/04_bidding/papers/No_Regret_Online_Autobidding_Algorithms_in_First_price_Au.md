# No-Regret Online Autobidding Algorithms in First-price Auctions

> 来源：arXiv 2510.16869 | 年份：2025 | 领域：ads/04_bidding（在线学习/拍卖理论）

## 问题定义

一价拍卖（First-price Auction）中赢家支付自己的出价（而非二价拍卖的第二高价），出价策略更复杂——需要 bid shading（低于真实估值出价）。本文设计具有 **No-Regret 保证** 的在线自动出价算法。

**与二价拍卖的关键区别**：
- 二价拍卖：诚实出价是占优策略（简单）
- 一价拍卖：最优出价 = 估值 - 信息租金（取决于竞争分布），策略性更强

## 模型结构图

```
┌──────────────────────────────────────────────────────┐
│              在线自动出价循环                           │
│                                                      │
│  Round t = 1, 2, ..., T                              │
│  ┌────────────────────────────────────────────────┐  │
│  │  1. 观测广告请求特征 x_t                        │  │
│  │  2. 估计价值 v_t = Value(x_t)                   │  │
│  │  3. 查询对偶变量 λ_t, μ_t (预算/ROI乘子)       │  │
│  │  4. 计算出价:                                   │  │
│  │     bid_t = v_t - λ_t/μ_t (bid shading)         │  │
│  │  5. 参与一价拍卖                                 │  │
│  │  6. 观测结果: win/lose, payment (if win)         │  │
│  │  7. 更新对偶变量 (在线梯度下降):                  │  │
│  │     λ ← λ + η(CPA - target)                     │  │
│  │     μ ← μ + η(cost - budget/T)                  │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
│  Regret = V*(最优固定策略) - V(在线算法)              │
│  保证: Regret = O(√T) → 平均遗憾 → 0                 │
└──────────────────────────────────────────────────────┘
```

## 核心方法与完整公式

### 公式1：一价拍卖最优出价（Bid Shading）

$$
\text{bid}^* = v - \frac{F(v)}{f(v)}
$$

**解释：**
- $v$：广告主对该展示的真实估值
- $F(v)$：竞争对手最高出价的 CDF
- $f(v)$：竞争对手最高出价的 PDF
- $F(v)/f(v)$：信息租金（hazard rate 的倒数）
- 需要知道竞争对手出价分布（实际中需在线估计）

### 公式2：在线凸优化框架

$$
\min_{\lambda, \mu \geq 0} \sum_{t=1}^{T} \ell_t(\lambda, \mu)
$$

$$
\ell_t(\lambda, \mu) = -(v_t - \text{payment}}_{\text{t) \cdot \mathbb{1}}[\text{win}}_{\text{t] + \lambda(\text{CPA}}_t - \text{target}) + \mu(\text{cost}}_{\text{t - B/T)
$$

**解释：**
- 对偶问题：通过优化拉格朗日乘子间接优化出价策略
- 在线梯度下降更新乘子，每步复杂度 $O(1)$

### 公式3：Regret Bound

$$
\text{Regret}}(T) \leq O(\sqrt{T \log T})
$$

**解释：**
- 平均遗憾 $\text{Regret}(T)/T = O(\sqrt{\log T / T}) \to 0$
- 一价拍卖中因策略梯度不连续，需平滑化技巧获得此 bound

### 公式4：自适应步长

$$
\eta_t = \frac{C}{\sqrt{\sum_{\tau=1}^{t} \|\nabla \ell_\tau\|^2}}
$$

**解释：** AdaGrad 风格步长，梯度大时步长小（稳定），梯度小时步长大（加速收敛）。

## 与基线方法对比

| 方法 | 理论保证 | 约束处理 | 实时性 | 实际效果 |
|------|---------|---------|--------|---------|
| **贪心出价** | 无 | 无 | 极快 | 差 |
| **固定 shading** | 无 | 简单 | 极快 | 中等 |
| **本文 (OCO)** | O(√T) | Lagrangian | 快 | 好 |
| **RL (PPO)** | 无理论保证 | Lagrangian | 快 | 好 |

## 实验结论

- 理论证明 O(√T) 遗憾界
- 1000+ 轮拍卖后效果接近最优离线策略的 95%+
- vs 贪心策略：效果提升 10-15%（同预算）
- 预算利用率提升约 5%
- 在非平稳环境中（竞争变化），自适应步长比固定步长好约 8%

## 工程落地要点

1. **毫秒级响应**：算法需简化为查表或线性运算，在线更新乘子 + 查表出价
2. **预算追踪**：实时追踪当天已花预算，动态调整出价系数
3. **竞争分布估计**：维护竞争对手出价分布的在线估计（指数滑动平均）
4. **风险控制**：设出价上下限防异常，避免过度竞价

## 面试考点

**Q1：一价拍卖和二价拍卖的区别及对出价策略的影响？**
> 二价拍卖赢家支付第二高价，诚实出价是占优策略。一价拍卖赢家支付自己出价，需 bid shading（低于真实估值出价），策略取决于竞争对手出价分布估计。

**Q2：什么是在线学习中的"遗憾"（Regret）？**
> Regret = 最优固定策略累计收益 - 在线算法累计收益。No-Regret 算法保证平均遗憾 → 0，表明算法长期表现接近最优。常见界：O(√T)。

**Q3：为什么一价拍卖越来越流行（取代二价拍卖）？**
> Header Bidding 兴起后，多个 SSP 同时竞价，已不是单一拍卖环境。一价拍卖更简单透明（出价即支付），避免二价拍卖在多层嵌套拍卖中的复杂性。Google AdX 于 2019 年转为一价拍卖。

**Q4：Lagrangian 对偶方法在出价约束中的优势？**
> 将硬约束（预算/CPA 上限）转为软约束，乘子自动学习约束的"价格"。预算紧张时乘子升高 → 出价降低；预算充裕时乘子降低 → 出价提高。无需手动设定出价规则。

**Q5：非平稳竞争环境下如何保持 No-Regret？**
> ① 使用 sliding window regret（只与最近 W 轮最优策略比较）② 自适应步长（对梯度变化敏感）③ 定期重置乘子（防止历史偏差累积）④ 检测环境变化的 change detection 机制。
