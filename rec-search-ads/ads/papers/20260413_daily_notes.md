# 广告系统论文/资源笔记 — 2026-04-13

## 1. Google Meridian: 开源营销组合建模框架

**来源：** https://github.com/google/meridian
**领域：** 营销组合建模（MMM）/ 广告效果衡量
**核心定位：** Google 构建的开源 MMM 框架，帮助营销人员做出数据驱动的决策

**核心技术：**
- 基于贝叶斯因果推断：混合先验知识与真实数据，揭示营销的真正增量影响
- 高度可定制的建模框架
- 完全开源，代码和方法论透明可审查

**2025 更新：**
- 支持非媒体变量（定价、促销）
- 渠道级贡献先验：营销人员可注入业务知识
- 上层漏斗媒体长期效果衡量（品牌回忆、延迟影响）
- 20+ 认证合作伙伴

**面试考点：** MMM vs 归因模型、贝叶斯因果推断在广告中的应用、增量效果衡量方法

## 2. LightGBM for Ranking: 梯度提升树排序

**来源：** https://lightgbm.readthedocs.io
**领域：** 排序学习 / 广告 CTR 排序
**核心定位：** 微软开发的高性能梯度提升树框架，支持回归、分类、排序

**排序能力：**
- LGBMRanker：专为 Learning-to-Rank 任务设计
- 使用 LambdaRank 梯度实现 LambdaMART 算法
- 分组训练：按 query group 组织训练数据

**核心优势：**
- 处理表格数据性能持续优于深度学习方法
- 广泛用于搜索引擎、推荐系统、电商平台的排序逻辑

**面试考点：** LambdaMART 原理、NDCG/MAP 指标优化、GBDT vs 深度学习在排序中的对比
