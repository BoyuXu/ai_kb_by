# GenCI: Generative Modeling of User Interest Shift via Cohort-based Intent Learning for CTR

> 来源：arXiv 2601.20345 | 领域：ads | 学习日期：20260402

## 问题定义

用户兴趣随时间动态变化（兴趣漂移），传统 CTR 模型难以捕捉这种时序动态。GenCI 通过基于用户群组的意图学习来生成式建模兴趣漂移，提升 CTR 预估的时效性。

## 核心方法与创新点

1. **群组意图学习**：将用户按行为模式聚类为群组（cohort），学习群组级别的意图转移模式
2. **生成式漂移建模**：用条件生成模型预测用户下一时段的兴趣分布，而非固定兴趣表征
3. **时序注意力**：设计时间感知的注意力机制，对近期行为赋予更高权重但保留长期偏好
4. **意图诊断**：生成的兴趣分布可解释，支持理解"用户为什么对这个广告感兴趣"

## 实验结论

- CTR 预估 AUC 提升 0.7%，对兴趣变化剧烈的用户提升更显著（1.2%）
- 群组级建模比个体级建模更稳定，方差降低 35%
- 兴趣漂移预测准确率 73%

## 工程落地要点

- **群组更新**：用户群组分配需要定期更新，建议小时级
- **实时特征**：兴趣漂移建模需要实时用户行为流数据
- **冷启动**：新用户先分配到相似群组，积累行为后个性化

## 面试考点

1. **Q：为什么用群组而不是个体建模兴趣漂移？**
   A：个体行为数据稀疏，兴趣漂移模式难以从单个用户学习。群组聚合后数据更充分，漂移模式更稳定。
2. **Q：生成式建模 vs 判别式建模兴趣的区别？**
   A：判别式直接预测用户是否会点击，生成式先预测用户的兴趣分布再匹配广告，提供更丰富的中间表征。
