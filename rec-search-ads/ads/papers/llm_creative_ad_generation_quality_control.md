# LLM-based Creative Ad Generation with Quality Control

> 来源：arXiv 2503.08421 | 领域：ads | 学习日期：20260402

## 问题定义

广告创意（文案 + 图片）的生产依赖人工设计，成本高、周期长、难以大规模个性化。LLM 驱动的自动生成可以大幅降低成本，但面临质量控制（合规性、品牌一致性、CTR 效果）挑战。

## 核心方法与创新点

1. **多阶段生成 Pipeline**：策略分析 → 创意生成 → 质量审核 → 效果预估，每阶段都有 LLM 参与
2. **约束解码**：在生成时通过 constrained decoding 确保广告文案满足长度、关键词、合规等约束
3. **CTR 预估引导**：用训练好的 CTR 预估模型对生成的创意打分，通过 RLHF 微调 LLM 使其倾向生成高 CTR 创意
4. **多模态生成**：文案和图片联合生成，确保视觉和文字的一致性

## 实验结论

- 自动生成创意的平均 CTR 达到人工创意的 92%，部分类目超越人工
- 创意生产效率提升 50 倍，成本降低 90%
- 合规审核通过率 97%（人工创意 95%）

## 工程落地要点

- **合规检查**：广告法合规是硬性要求，需要专门的审核模块
- **A/B 测试**：自动创意需要与人工创意进行充分的在线对比
- **品牌一致性**：需要维护品牌风格指南作为 LLM 的 prompt context

## 常见考点

1. **Q：为什么 CTR 引导生成比直接优化好？**
   A：直接优化可能导致标题党式的创意，CTR 预估模型作为奖励信号可以结合合规约束综合优化。
2. **Q：广告创意生成的评估指标？**
   A：CTR、CVR、ROAS（广告支出回报率）、合规率、多样性。

## 创意生成架构详解

### 生成管线
- **输入理解**：商品属性解析 + 广告主意图理解 + 目标人群分析
- **内容生成**：LLM 生成文案 / 扩散模型生成图片 / 视频模板渲染
- **质量过滤**：CTR 预估打分 + 品牌安全检测 + 多样性去重

### 多模态模型
- 文本生成：GPT/LLaMA 风格 LLM + 商品属性 prompt
- 图像生成：Stable Diffusion + ControlNet + 商品图融合
- 视频生成：模板渲染 + 动态元素替换

### 在线优化
- Contextual Bandit：根据用户画像选择最优创意变体
- 多臂老虎机：Thompson Sampling / UCB 选择创意
- 反馈闭环：展示数据回流更新选择策略和生成模型

## 面试深度追问

- **Q: 广告创意的 CTR 预估和普通推荐有什么不同？**
  A: 1) 创意特征多模态（文本+图片+视频）；2) 同一商品多个创意变体需要对比评估；3) 创意疲劳效应（同一创意重复展示 CTR 衰减）。

- **Q: 生成创意的品牌安全如何保障？**
  A: 多层防线：1) Prompt 约束（禁止生成敏感内容）；2) 后处理过滤（敏感词检测/图像安全分类）；3) 人工审核抽检（高曝光创意全量审核）。

- **Q: 动态创意优化（DCO）的技术方案？**
  A: 将创意拆解为元素（标题/图片/CTA/背景色），每个元素多个变体，通过 Bandit 算法在线选择最优组合。组合空间爆炸问题用分层 Bandit 或 Thompson Sampling 解决。

- **Q: 如何衡量创意对 GMV 的增量贡献？**
  A: 因果推断方法：1) A/B Test（新创意 vs 旧创意）；2) PSM（倾向得分匹配）控制混杂因素；3) 双重差分法（DID）评估长期效果。
