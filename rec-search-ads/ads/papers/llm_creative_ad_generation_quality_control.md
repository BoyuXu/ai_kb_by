# LLM-based Creative Ad Generation with Quality Control

> 来源：arXiv 2503.08421 | 领域：ads | 学习日期：20260402

## 问题定义

广告创意（文案 + 图片）的生产依赖人工设计，成本高、周期长、难以大规模个性化。LLM 驱动的自动生成可以大幅降低成本，但面临质量控制（合规性、品牌一致性、CTR 效果）挑战。

## 核心方法与创新点

1. **多阶段生成 Pipeline**：策略分析 → 创意生成 → 质量审核 → 效果预估，每阶段都有 LLM 参与
2. **约束解码**：在生成时通过 constrained decoding 确保广告文案满足长度、关键词、合规等约束
3. **CTR 预估引导**：用训练好的 CTR 预估模型对生成的创意打分，通过 RLHF 微调 LLM 使其倾向生成高 CTR 创意
4. **多模态生成**：文案和图片联合生成，确保视觉和文字的一致性

## 实验结论

- 自动生成创意的平均 CTR 达到人工创意的 92%，部分类目超越人工
- 创意生产效率提升 50 倍，成本降低 90%
- 合规审核通过率 97%（人工创意 95%）

## 工程落地要点

- **合规检查**：广告法合规是硬性要求，需要专门的审核模块
- **A/B 测试**：自动创意需要与人工创意进行充分的在线对比
- **品牌一致性**：需要维护品牌风格指南作为 LLM 的 prompt context

## 常见考点

1. **Q：为什么 CTR 引导生成比直接优化好？**
   A：直接优化可能导致标题党式的创意，CTR 预估模型作为奖励信号可以结合合规约束综合优化。
2. **Q：广告创意生成的评估指标？**
   A：CTR、CVR、ROAS（广告支出回报率）、合规率、多样性。
