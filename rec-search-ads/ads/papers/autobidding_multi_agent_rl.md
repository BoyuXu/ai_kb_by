# AutoBidding with Multi-agent Reinforcement Learning in Display Advertising

> 来源：arXiv 2501.12399 | 领域：ads | 学习日期：20260402

## 问题定义

展示广告的自动出价面临多广告主竞争博弈问题：每个广告主的出价策略都会影响其他广告主的竞价环境。单智能体 RL 忽略了多方博弈动态，多智能体 RL 方法可以更好地建模竞争环境下的出价策略。

## 核心方法与创新点

1. **多智能体建模**：每个广告主的出价策略作为一个独立 agent，通过 MARL 学习竞争均衡策略
2. **中心化训练-去中心化执行（CTDE）**：训练时可以访问所有 agent 的信息，执行时各 agent 独立决策
3. **预算约束**：在 RL 框架中引入预算约束作为 Lagrangian 松弛项，动态调整出价上限
4. **市场模拟器**：构建离线竞价模拟器用于策略训练和评估，避免在线探索的成本

## 实验结论

- 相比单智能体 RL，广告主平均 ROI 提升 8.5%
- 平台整体广告收入提升 3.2%
- 竞价市场的纳什均衡收敛速度提升 40%

## 工程落地要点

- **模拟器保真度**：竞价模拟器需要高精度建模真实的竞价动态
- **隐私保护**：CTDE 训练需要跨广告主数据，需要联邦学习或差分隐私保护
- **策略更新频率**：agent 策略需要定期更新以适应市场变化

## 常见考点

1. **Q：为什么用多智能体而不是单智能体 RL？**
   A：竞价是多方博弈，单智能体将其他广告主视为环境的一部分，无法捕捉策略交互和博弈均衡。
2. **Q：CTDE 的优势是什么？**
   A：训练时利用全局信息学习更好的策略，执行时只需本地信息即可决策，兼顾效果和可部署性。
3. **Q：预算约束如何融入 RL？**
   A：通过 Lagrangian 松弛将约束优化转化为无约束优化，Lagrange 乘子自动调整出价上限。
