# RQ-GMM: Residual Quantized Gaussian Mixture for Multimodal Semantic Discretization in CTR

> 来源：arXiv 2602.04321 | 领域：ads | 学习日期：20260402

## 问题定义

多模态内容特征（文本、图片、视频）在 CTR 预估中的融合面临语义鸿沟：连续的多模态嵌入与 CTR 模型的离散特征空间不匹配。RQ-GMM 通过残差量化将多模态语义离散化，使其与传统 ID 特征无缝融合。

## 核心方法与创新点

1. **高斯混合量化**：用 GMM 将多模态嵌入聚类为语义簇，每个簇对应一个离散的语义 ID
2. **残差量化**：多级量化逐步细化语义粒度——第一级捕捉大类语义，后续级别捕捉细粒度差异
3. **端到端训练**：量化模块与 CTR 模型联合训练，语义 ID 的分配直接受 CTR 目标驱动
4. **跨模态对齐**：文本和图片的语义 ID 在同一空间中，支持跨模态特征交叉

## 实验结论

- 多模态语义 ID 融入 CTR 模型后 AUC 提升 0.6%
- 相比直接使用连续嵌入，离散化后的特征交叉效果更好（AUC +0.3%）
- 残差量化的多级语义比单级量化效果提升 15%

## 工程落地要点

- **码本大小**：每级量化的码本大小需要调优，太小损失信息，太大增加稀疏性
- **在线推理**：多模态特征预计算并离散化后存储，推理时直接查表
- **码本更新**：随着新内容上线，需要定期更新 GMM 聚类中心

## 面试考点

1. **Q：为什么要把多模态特征离散化？**
   A：CTR 模型擅长处理离散特征的交叉，连续嵌入难以与 ID 类特征有效交叉。离散化后可以复用成熟的特征交叉技术。
2. **Q：残差量化 vs 向量量化的区别？**
   A：向量量化一步到位，残差量化逐级细化——先粗后细，信息损失更小，表达力更强。
