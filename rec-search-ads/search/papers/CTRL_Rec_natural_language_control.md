# CTRL-Rec: Controlling Recommender Systems With Natural Language

**ArXiv:** 2510.12742 | **Date:** 2025-10

## 核心问题
用户难以用自然语言实时控制推荐系统，传统推荐缺乏透明度和用户可解释性。

## 核心方案
允许用户通过自然语言请求实时引导推荐，例如："我想看不同视角的尊重性内容"。

## 技术方案
### 训练阶段
用 LLM 模拟用户对候选内容的判断（基于其语言请求），训练近似此类判断的嵌入模型。

### 部署阶段
每次用户请求仅需一次 LLM embedding 计算，实现实时控制，计算高效。

## 核心优势
- 实时可控：用户可随时用自然语言更改推荐偏好
- 计算高效：部署时无需完整 LLM 推理
- 透明度：用户能理解和控制推荐逻辑

## 对比 CRS（对话推荐系统）
CTRL-Rec 更轻量，无需多轮对话，单次语言请求即可修改推荐行为。
