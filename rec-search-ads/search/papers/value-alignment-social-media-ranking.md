# Value Alignment of Social Media Ranking Algorithms

- **Date**: 2025-09 (revised 2026-03)
- **Domain**: Search/Ranking
- **URL**: https://arxiv.org/abs/2509.14434

## 核心贡献

提出基于Schwartz基本人类价值理论的社交媒体Feed价值对齐方法。实现了用户可通过控制器表达价值权重，组合权重与帖子价值表达进行排序。

## 关键技术

- 基于Schwartz人类价值理论建模社交媒体内容
- 用户可配置的价值权重控制器
- 价值表达与排序得分的融合机制

## 实验结果

- N=141和N=250的对照实验验证了用户可以使用控制器构建反映期望价值的Feed
- Engagement导向排序可能优先特定个人主义价值，导致不期望的社会结果

## 面试考点

- 推荐系统中的价值对齐问题
- 用户控制与算法推荐的平衡
- 社交媒体排序中的公平性考量
