# Large Language Model Enhanced Graph Invariant Contrastive Learning for Out-of-Distribution Recommendation

> 来源：arXiv 2025 | 领域：rec-sys | 学习日期：20260404

## 问题定义

推荐系统在分布外（OOD）场景泛化差：
- **时序 OOD**：训练期 vs 测试期用户行为分布漂移
- **地域 OOD**：A 城市模型在 B 城市效果差
- **人群 OOD**：主流用户模型无法泛化到长尾用户

核心挑战：如何学习 **不变特征（Invariant Features）**，在不同分布下保持预测能力？

## 核心方法与创新点

1. **LLM 增强图构建**：
   - 用 LLM 分析物品文本，提取语义不变特征（品类、功能、目标人群）
   - 构建语义图：相同功能/品类的物品连边
   
$$\mathcal{G}_{\text{sem}} = \{(i, j) : \text{sem\_sim}(e_i^{\text{LLM}}, e_j^{\text{LLM}}) > \theta\}$$

2. **图不变学习（Graph Invariant Learning, GIL）**：
   - 将图分为 **环境子图**（分布特定）和 **不变子图**（跨分布稳定）
   - 训练目标：不变子图特征在多个环境下预测能力一致
   
$$\min_\phi \max_e \mathcal{L}(f(\mathcal{G}^{\text{inv}}_\phi), y^e)$$

3. **对比不变性增强**：
   - 同物品在不同环境（不同用户群/时间段）的表示应相近
   
$$\mathcal{L}_{\text{inv}} = -\log \frac{\exp(\text{sim}(z_i^{e1}, z_i^{e2})/\tau)}{\sum_j \exp(\text{sim}(z_i^{e1}, z_j^{e2})/\tau)}$$

4. **LLM 作为环境生成器**：
   - LLM 自动划分"环境"（时间段、用户群）
   - 不需要人工定义分布划分

## 实验结论

- 跨地域泛化: NDCG@10 **+17.3%** vs 普通图推荐
- 跨时段泛化: Recall@20 **+12.1%**
- 长尾用户场景: Hit Rate **+22.6%**

## 工程落地要点

- LLM 语义图离线构建，每周更新（新品入库时）
- 环境划分建议：时间（月份/季节）+ 用户活跃度（高/中/低）
- GIL 训练额外计算约 2x，但推理无额外开销
- 不变子图大小超参：建议 30-50% 的边为不变边

## 面试考点

1. **Q**: 什么是推荐系统的 OOD 问题？  
   **A**: 模型在训练分布外的场景（新时间段、新地域、新用户群）性能显著下降，因为学到了与分布相关的虚假特征。

2. **Q**: 图不变学习如何区分不变特征和环境特征？  
   **A**: 通过多环境训练：不变特征在所有环境下保持预测能力；环境特征在特定环境有效但跨环境失效。用 IRM（不变风险最小化）或 GIL 识别。

3. **Q**: LLM 在这里扮演什么角色？  
   **A**: 两个角色：(1) 提取物品语义不变特征（功能/品类），比 ID 特征更稳定；(2) 自动划分环境，替代人工定义分布。
