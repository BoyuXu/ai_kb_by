# NVIDIA Merlin NVTabular

- **Type**: GPU加速特征工程库
- **URL**: https://github.com/NVIDIA-Merlin/NVTabular

## 核心特性

- **95x加速**: 多GPU DGX A100 vs Spark CPU集群
- **TB级处理**: 超越GPU/CPU内存限制
- **端到端流水线**: NVTabular(特征) → HugeCTR(训练) → Triton(推理)
- **框架集成**: PyTorch/TensorFlow数据加载器

## 面试考点

GPU加速数据处理，大规模推荐系统数据流水线，RAPIDS生态。
