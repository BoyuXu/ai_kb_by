# Transformers4Rec: NVIDIA Session-based Recommendation

- **Type**: Open-Source Library
- **URL**: https://nvidia-merlin.github.io/Transformers4Rec/

## 核心特性

- **序列推荐**: Session-based next-item prediction
- **HuggingFace集成**: 复用NLP Transformer架构
- **GPU加速**: NVIDIA Merlin生态组件
- **端到端**: NVTabular → Transformers4Rec → Triton

## 面试考点

Session-based推荐，Transformer在推荐中的应用，NLP与RecSys的桥接。
