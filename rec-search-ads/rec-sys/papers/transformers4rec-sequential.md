# NVIDIA Transformers4Rec: Transformers for Sequential Recommendation

- **Type**: Open-Source Library
- **URL**: https://github.com/NVIDIA-Merlin/Transformers4Rec

## 核心特性

- 桥接NLP和RecSys：集成HuggingFace Transformers
- 模块化构建块，支持多塔、多头/多任务架构
- 序列和会话推荐场景

## 面试考点

Transformer在序列推荐中的应用，session-based推荐的建模方法。
