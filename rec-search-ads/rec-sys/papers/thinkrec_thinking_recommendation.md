# ThinkRec: Thinking-based Recommendation via LLM

- **Type**: Research Paper
- **URL**: https://arxiv.org/abs/2505.15091

## 核心贡献

从快思考转向慢思考的推荐框架，通过推理链增强LLM推荐的准确性和可解释性。

## 关键技术

- **思考激活机制**: 关键词摘要 + 合成推理链
- **实例级专家融合**: 动态权重分配
- **推理链注入**: 增强个性化特征捕获

## 面试考点

LLM4Rec推理范式，可解释推荐，System 1 vs System 2思维。
