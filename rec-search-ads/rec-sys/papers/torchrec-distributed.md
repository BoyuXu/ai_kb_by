# TorchRec: PyTorch Distributed Recommendation Training

- **Type**: Open-Source Library
- **URL**: https://github.com/meta-pytorch/torchrec

## 核心特性

Meta开源，支持大规模embedding表跨GPU分片。Sharding策略：data-parallel, table-wise, row-wise, column-wise等。Fully Sharded 2D Parallelism支持。

## 面试考点

Embedding表的分布式切分策略，推荐模型的混合并行训练。
