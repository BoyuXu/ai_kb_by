# RecBole v2.0: Unified Recommendation Library

- **Type**: Open-Source Framework
- **URL**: https://github.com/RUCAIBox/RecBole

## 核心特性

- **100+算法**: Python/PyTorch统一实现
- **8个特化包**: 数据稀疏、偏差、分布偏移、GNN、Transformer
- **44个数据集**: 标准化格式与评估
- **公平性**: 去偏学习、公平推荐模块
